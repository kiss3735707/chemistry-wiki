# 物质的量 · 沪科版一页复习

## 核心公式

| 公式 | 适用 |
|------|------|
| n = N / N_A | 粒子数换算 |
| n = m / M | 质量换算 |
| n = V / 22.4 | **标况气体**（V 单位 L） |
| c = n / V | 溶液浓度（V 为溶液体积 L） |
| c₁V₁ = c₂V₂ | 稀释 |

**M 数值 = 相对分子/原子质量（g/mol）**

---

## 换算关系图

```
m ←─M─→ n ─N_A→ N
         ↓
    22.4（标况气）→ V
         ↓
       c = n/V
```

---

## 六类必会题型

1. **化学式粒子关系**：0.5 mol H₂SO₄ → H 1 mol、O 2 mol  
2. **质量计算**：88 g CO₂ → n = 2 mol  
3. **标况气体**：33.6 L CO₂ → n = 1.5 mol  
4. **稀释**：20 mL 5 mol/L → 500 mL → c₂ = 0.2 mol/L  
5. **方程式综合**：5.6 g Fe → V(H₂) = 2.24 L（标况）  
6. **密度求 M**：M = 22.4 × ρ（g/L，标况气体）

---

## 易错三句话

1. **22.4 只给标况气体用**，H₂O、乙醇不能用  
2. **c = n/V 的 V 是溶液体积**，不是溶剂体积  
3. **1 mol O₂ 含 2 mol O 原子**，64 g O₂ = 4 mol O 原子
