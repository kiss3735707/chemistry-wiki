# 复分解反应何时产生沉淀

> 综合 [[concepts/复分解反应]]、[[concepts/溶解度与溶解性规律]]、[[summaries/初中化学溶解性表]] 整理。

## 核心结论

复分解反应 **AB + CD → AD + CB** 要能发生，生成物中至少满足一条：**有水、有气体、有沉淀**。

**有沉淀** = 交换成分后，新生成的某种化合物在水中**难溶或微溶**，从溶液中析出。

## 判断三步法

1. 写出交换后的两种生成物
2. 用溶解性口诀判断能否溶解
3. 至少一种**难溶或微溶** → 有沉淀 → 反应可发生

**初中约定：微溶也算沉淀**（CaSO₄、Ag₂SO₄、PbCl₂、Ca(OH)₂）。

## 必背口诀

```
钾钠铵硝皆易溶
氯除银铅
硫除钡铅钙
碳磷碳酸多不溶
碱除钾钠钡钙微
```

| 口诀 | 典型沉淀 |
|------|----------|
| 氯除银铅 | AgCl↓ 白；PbCl₂ 微溶 |
| 硫除钡铅钙 | BaSO₄↓ 白；CaSO₄ 微溶 |
| 碳磷碳酸多不溶 | CaCO₃↓、BaCO₃↓ 等 |
| 碱除钾钠钡钙微 | Cu(OH)₂↓ 蓝、Fe(OH)₃↓ 红褐 |

## 常见有沉淀的反应

| 反应 | 沉淀 |
|------|------|
| BaCl₂ + Na₂SO₄ → BaSO₄↓ + 2NaCl | 白色 BaSO₄（检验 SO₄²⁻） |
| AgNO₃ + NaCl → AgCl↓ + NaNO₃ | 白色 AgCl（检验 Cl⁻） |
| 2NaOH + CuSO₄ → Cu(OH)₂↓ + Na₂SO₄ | 蓝色 Cu(OH)₂ |
| 3NaOH + FeCl₃ → Fe(OH)₃↓ + 3NaCl | 红褐色 Fe(OH)₃ |

## 没沉淀也能反应的情况

复分解不要求必须有沉淀，满足以下任一即可：

- **生成水** → [[concepts/中和反应]]（HCl + NaOH → NaCl + H₂O）
- **生成气体** → Na₂CO₃ + 2HCl → 2NaCl + H₂O + CO₂↑
- **生成沉淀** → 见上表

## 易错点

1. **AgCl、BaSO₄ 不溶于稀酸** → Cl⁻、SO₄²⁻ 专属检验；CaCO₃ 加酸溶解冒泡，可区分
2. **Na₂CO₃ 是盐不是碱**，与酸反应出 CO₂ 仍属复分解
3. 生成物全是 K⁺/Na⁺/NH₄⁺/NO₃⁻ 的可溶盐，且无水无气 → **通常不反应**（如 NaCl + KNO₃）

## 快速练习

**CuSO₄ + NaOH → ?**  
→ Cu(OH)₂ 难溶 + Na₂SO₄ 易溶 → **有蓝色沉淀，能反应**

**NaCl + KNO₃ → ?**  
→ NaNO₃、KCl 均可溶，无水无气 → **无沉淀，不反应**
