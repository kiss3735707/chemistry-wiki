# 复分解反应

## 定义

两种化合物互相交换成分，生成另外两种化合物的反应。

**通式：** AB + CD → AD + CB

## 发生条件

满足**至少一条**即可发生：
- 生成**水**
- 生成**气体**
- 生成**沉淀**

## 溶解性判断

复分解能否产生沉淀，需查 [[concepts/溶解度与溶解性规律]]：

```
钾钠铵硝皆易溶；氯除银铅；硫除钡铅钙；碳磷碳酸多不溶
```

**微溶视为沉淀**：CaSO₄、Ag₂SO₄、PbCl₂、Ca(OH)₂

## 典型例子

- 中和：HCl + NaOH → NaCl + H₂O（生成水）
- 检验 CO₃²⁻：CaCO₃ + 2HCl → CaCl₂ + H₂O + CO₂↑（生成气体）
- 检验 SO₄²⁻：BaCl₂ + H₂SO₄ → BaSO₄↓ + 2HCl（生成沉淀）

## 易错点

- Na₂CO₃ 是**盐**不是碱，但溶液呈碱性
- AgCl、BaSO₄ 不溶于稀酸，可用于专属离子检验
