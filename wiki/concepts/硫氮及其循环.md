## 3.1 硫及其重要化合物

### 一、概念

#### 1. 硫单质

| 项目 | 内容 |
|------|------|
| 物理性质 | 黄色固体，难溶于水，微溶于酒精，易溶于 CS₂ |
| 化学性质 | 与 O₂、Fe、Cu、H₂ 等反应；既有氧化性又有还原性 |
| 存在 | 火山口、硫铁矿（FeS₂）等 |

**典型反应：**

```
S + O₂ → SO₂              （点燃，淡蓝色火焰）
Fe + S → FeS              （加热，黑色固体）
H₂ + S → H₂S              （加热）
```

#### 2. 二氧化硫 SO₂

详见 [[entities/二氧化硫]]。

| 项目 | 内容 |
|------|------|
| 物理 | 无色、有刺激性；易溶于水（1:40）；密度 > 空气 |
| 酸性氧化物 | SO₂ + H₂O ⇌ H₂SO₃ |
| **漂白性** | 与有色物质**化合**，加热可恢复原色 |
| 还原性 | SO₂ + O₂ → SO₃（催化剂）；可被 KMnO₄ 氧化 |
| 环境 | 酸雨前体 |

#### 3. 三氧化硫 SO₃

- 易溶于水：SO₃ + H₂O → H₂SO₄
- 工业制硫酸中间产物：2SO₂ + O₂ ⇌ 2SO₃（V₂O₅ 催化）→ 吸收制 H₂SO₄

#### 4. 硫酸 H₂SO₄

详见 [[entities/硫酸]]。

**稀硫酸：** 体现酸的通性（H⁺），与活泼金属、碱、碱性氧化物、某些盐反应。

**浓硫酸三大特性：**

| 特性 | 本质 | 实例 |
|------|------|------|
| **吸水性** | 吸收游离水 | 干燥 H₂、CO₂；使蓝矾变白 |
| **脱水性** | 按 H:O=2:1 脱去有机物中的 H、O | 使蔗糖、棉花变黑炭化 |
| **强氧化性** | 加热时 +6 价 S 得电子 | 与 Cu、C 反应；**常温钝化** Fe、Al |

**浓硫酸与 Cu、C 反应（需加热）：**

```
Cu + 2H₂SO₄(浓) → CuSO₄ + SO₂↑ + 2H₂O
C + 2H₂SO₄(浓) → CO₂↑ + 2SO₂↑ + 2H₂O
```

**钝化：** 常温下 Fe、Al 与浓 H₂SO₄/浓 HNO₃ 接触表面生成致密氧化膜，阻止反应继续。

---

### 二、举例

**例 1**：SO₂ 使品红褪色，加热后恢复红色 → **化合型漂白**（对比 [[comparisons/SO₂漂白 vs HClO漂白]]）。

**例 2**：鉴别 SO₂ 与 CO₂——品红溶液（SO₂ 褪色）或酸性 KMnO₄（SO₂ 使其褪色）。

**例 3**：浓硫酸使 CuSO₄·5H₂O 变白色 → **吸水性**（不是脱水性，结晶水属于「组成水」）。

**例 4**：浓硫酸滴入蔗糖 → 变黑膨胀 → **脱水性**。

**例 5**：测定硫酸铜晶体中结晶水（本章活动）

```
CuSO₄·xH₂O → CuSO₄ + xH₂O↑    （加热至恒重）
x = n(H₂O)/n(CuSO₄) = Δm(H₂O)/18 ÷ m(CuSO₄)/160
```

---

### 三、拓展

- **工业制硫酸（接触法）**：S 或 FeS₂ → SO₂ → SO₃ → H₂SO₄
- **SO₂ 的还原性**：SO₂ + Cl₂ + 2H₂O → H₂SO₄ + 2HCl
- **H₂S**（了解）：有臭鸡蛋气味，还原性气体，SO₂ + 2H₂S → 3S + 2H₂O

---

### 四、易错点（3.1）

| 说法 | 正误 | 解析 |
|------|------|------|
| SO₂ 使紫色石蕊先变红后褪色 | ⚠️ | 变红是酸性；**褪色**需用**品红**等特定试剂 |
| SO₂ 漂白所有有色物质 | ❌ | 不能漂白酸碱指示剂 |
| 浓硫酸使蓝矾变白是脱水性 | ❌ | 是**吸水性**（吸收结晶水） |
| 浓硫酸与 Cu 反应不需加热 | ❌ | 需**加热**才体现强氧化性 |
| 稀硫酸与 Cu 反应 | ❌ | Cu 在 H⁺ 后，不反应 |

## 3.2 氮及其重要化合物

### 一、概念

#### 1. 氮气 N₂

| 项目 | 内容 |
|------|------|
| 物理 | 无色无味气体，密度略小于空气 |
| 化学 | **很稳定**（N≡N 三键）；高温/闪电下与 O₂ 反应 |
| 用途 | 保护气、液氮；工业合成氨原料 |

```
N₂ + O₂ → 2NO              （放电或高温，自然固氮）
N₂ + 3H₂ ⇌ 2NH₃           （高温高压、催化剂，工业合成氨）
```

#### 2. 氨 NH₃

详见 [[entities/氨]]。

| 项目 | 内容 |
|------|------|
| 物理 | 极易溶于水（1:700）；密度 < 空气 |
| 化学 | **碱性**；NH₃ + HCl → NH₄Cl（白烟） |
| 喷泉实验 | 利用 NH₃ 极易溶于水，烧瓶内压强骤降，水被压入形成喷泉 |

#### 3. 铵盐

- 与碱共热放出 NH₃：**检验 NH₄⁺**
- 受热分解：NH₄Cl ⇌ NH₃ + HCl；NH₄HCO₃ → NH₃ + CO₂ + H₂O

```
NH₄⁺ + OH⁻ → NH₃↑ + H₂O    （加热，湿润红色石蕊变蓝）
```

#### 4. 一氧化氮 NO 与 二氧化氮 NO₂

| 物质 | 颜色 | 要点 |
|------|------|------|
| **NO** | 无色 | 不与水反应；**遇 O₂ 变红棕色** |
| **NO₂** | 红棕色 | 有毒；3NO₂ + H₂O → 2HNO₃ + NO |

```
2NO + O₂ → 2NO₂
3NO₂ + H₂O → 2HNO₃ + NO
4NO₂ + O₂ + 2H₂O → 4HNO₃    （综合）
```

#### 5. 硝酸 HNO₃

详见 [[entities/硝酸]]。

| 项目 | 内容 |
|------|------|
| 强酸性 | 完全电离 |
| **强氧化性** | 浓/稀均具氧化性；与 Cu、Ag 等反应 |
| 钝化 | 常温 Fe、Al 遇浓 HNO₃ 钝化 |
| 见光分解 | 4HNO₃ → 4NO₂↑ + O₂↑ + 2H₂O（保存在棕色瓶） |

**Cu 与硝酸：**

```
3Cu + 8HNO₃(稀) → 3Cu(NO₃)₂ + 2NO↑ + 4H₂O
Cu + 4HNO₃(浓) → Cu(NO₃)₂ + 2NO₂↑ + 2H₂O
```

---

### 二、举例

**例 6**：实验室制 NH₃

```
2NH₄Cl + Ca(OH)₂ → CaCl₂ + 2NH₃↑ + 2H₂O    （加热，碱石灰干燥）
```

**例 7**：NO 的鉴定——无色气体接触空气变红棕色。

**例 8**：NO₂ 与水

```
3NO₂ + H₂O → 2HNO₃ + NO
```

1 mol NO₂ 与水反应，转移 2/3 mol 电子（N 从 +4 → +5 和 +2）。

**例 9**：检验 NH₄⁺——加 NaOH 加热，湿润红色石蕊试纸变蓝。

---

### 三、拓展

- **合成氨工业**：N₂ + 3H₂ ⇌ 2NH₃（高温、高压、铁催化剂）；勒夏特列原理应用
- **氮的固定**：人工固氮（合成氨）、生物固氮（根瘤菌）、大气固氮（闪电）
- **NO、NO₂ 转化**：NO₂ 既作氧化剂又作还原剂（歧化）

---

### 四、易错点（3.2）

| 说法 | 正误 | 解析 |
|------|------|------|
| N₂ 与 O₂ 在常温下大量反应 | ❌ | N₂ 很稳定，需放电或高温 |
| NH₃ 使紫色石蕊变蓝 | ✅ | 碱性气体 |
| NH₃ 使酚酞变红 | ✅ | 溶于水呈碱性 |
| NO 能用排空气法收集 | ❌ | 与 O₂ 反应，用**排水法** |
| 浓 HNO₃ 与 Cu 反应无 NO₂ | ❌ | 浓硝酸 → NO₂；稀硝酸 → NO |
| Fe 与过量稀 HNO₃ 反应生成 H₂ | ❌ | 硝酸有氧化性，**不生成 H₂** |

## 3.3 硫循环与氮循环

### 一、概念

#### 1. 硫循环（简化）

```
地壳/矿石（FeS₂ 等）→ SO₂（燃烧）→ H₂SO₄（酸雨）
    ↓
植物吸收 SO₄²⁻ → 食物链 → 分解归还
```

#### 2. 氮循环（简化）

```
N₂（大气）⇌ NH₃ / NH₄⁺（固氮）→ 蛋白质 → 分解 → NO₃⁻ → N₂
    ↓
NO、NO₂（汽车尾气、闪电）→ HNO₃（酸雨）
```

#### 3. 酸雨

| 类型 | 前体 | 主要产物 |
|------|------|----------|
| **硫酸型** | SO₂ | H₂SO₄ |
| **硝酸型** | NO、NO₂ | HNO₃ |

- **判定**：pH < **5.6** 的雨水为酸雨
- **危害**：腐蚀建筑、酸化土壤和水体、影响生态
- **防治**：燃煤脱硫、汽车尾气催化转化、使用清洁能源

---

### 二、举例

**例 10**：汽车尾气含 NO，经催化转化：2NO + 2CO → N₂ + 2CO₂

**例 11**：石灰石脱硫：CaCO₃ + SO₂ → CaSO₃ + CO₂（了解）

---

### 三、拓展

- **温室气体 vs 酸雨**：CO₂ 导致温室效应，不是酸雨主因
- **臭氧层**：与 NO_x 催化破坏有关（了解）

---

### 四、易错点（3.3）

| 说法 | 正误 | 解析 |
|------|------|------|
| 酸雨 pH = 7 | ❌ | 正常雨水 pH≈5.6（溶 CO₂）；酸雨 **< 5.6** |
| CO₂ 是形成酸雨的主要物质 | ❌ | 主要前体是 **SO₂** 和 **NO_x** |
| 酸雨只含硫酸 | ❌ | 可有硫酸型、硝酸型或混合型 |

---

## 五、本章速记卡片

```
SO₂：酸性氧化物 + 漂白（化合，可逆）+ 还原性
浓 H₂SO₄：吸水 / 脱水 / 强氧化（加热）；钝化 Fe、Al

N₂ 稳定 → 合成氨 N₂ + 3H₂ ⇌ 2NH₃
NH₃ 极易溶于水、碱性；NH₄⁺ 检验：碱 + 加热
NO 无色 → 遇 O₂ 变红棕色 NO₂
3NO₂ + H₂O → 2HNO₃ + NO

HNO₃：强氧化性；浓→NO₂，稀→NO；棕色瓶保存

酸雨：SO₂（硫酸型）、NO_x（硝酸型）；pH < 5.6
```
