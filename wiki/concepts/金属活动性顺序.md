# 金属活动性顺序

## 定义

按金属与酸、盐溶液等反应的**活泼程度**排列的顺序，是初中化学最重要的排序之一。

## 标准顺序

```
K → Ca → Na → Mg → Al → Zn → Fe → Sn → Pb → (H) → Cu → Hg → Ag → Pt → Au
```

**谐音：** 钾钙钠镁铝，锌铁锡铅氢，铜汞银铂金

## 应用规则

1. **前置换后**：活泼金属 + 不活泼金属盐溶液 → 置换（如 Fe + CuSO₄）
2. **H 前 + 稀酸 → H₂↑**：Zn + H₂SO₄ → ZnSO₄ + H₂↑
3. **H 后 + 稀酸 → 一般不反应**：Cu + HCl 不反应
4. **K、Ca、Na 太活泼**：不能从盐溶液置换（先与水反应）

## 易错点

- 地壳中 **Al 含量 > Fe**，但活动性 **Al > Fe**（含量 ≠ 活泼性）
- 铁生锈是缓慢氧化，铁在 O₂ 中燃烧为剧烈氧化
