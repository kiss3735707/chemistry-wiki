# 化学方程式计算

## 依据

**质量守恒定律**：反应前后各原子种类、数目、总质量不变。

## 计算步骤

1. 设未知数
2. 写化学方程式并配平
3. 标相对质量（或物质的量）
4. 列比例式
5. 求解并作答

## 常用相对分子质量

| 物质 | Mr |
|------|-----|
| H₂O | 18 |
| CO₂ | 44 |
| CaCO₃ | 100 |
| Ca(OH)₂ | 74 |

## 经典比例

- 电解水：H₂ : O₂ 体积 = **2:1**，H : O 质量 = **1:8**
- 2H₂ + O₂ → 2H₂O：H₂ : O₂ : H₂O = 4 : 32 : 36 = **1:8:9**

## 溶液结合计算

- 溶质质量 = 溶液质量 × 溶质质量分数
- 稀释：m₁ω₁ = m₂ω₂（溶质质量不变）
