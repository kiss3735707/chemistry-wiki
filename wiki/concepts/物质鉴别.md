# 物质鉴别

## 气体鉴别

| 气体 | 方法 |
|------|------|
| O₂ | 带火星木条**复燃** |
| CO₂ | 通入**澄清石灰水**变浑浊 |
| H₂ | 点燃，**尖锐爆鸣声**（须先验纯） |
| CO | 点燃后产物使澄清石灰水变浑浊 |

## 离子检验

| 离子 | 试剂 | 现象 |
|------|------|------|
| Cl⁻ | AgNO₃ + HNO₃ | 白色沉淀 AgCl（不溶于酸） |
| SO₄²⁻ | BaCl₂ + HCl | 白色沉淀 BaSO₄（不溶于酸） |
| CO₃²⁻ | 稀盐酸 | 气泡，石灰水变浑浊 |
| NH₄⁺ | NaOH 加热 | 刺激性气体，湿润石蕊变蓝 |
| Fe³⁺ | NaOH | 红褐色沉淀 Fe(OH)₃ |
| Cu²⁺ | NaOH | 蓝色沉淀 Cu(OH)₂ |

## 颜色鉴别速记

| 颜色 | 常见物质 |
|------|----------|
| 黑色固体 | CuO、Fe₃O₄、C、MnO₂ |
| 蓝色溶液 | CuSO₄、CuCl₂ |
| 浅绿色溶液 | FeSO₄、FeCl₂ |
| 黄色溶液 | FeCl₃ |
