# 物质的分类

> 沪科版高一必修第一册 · 第 1 章 · 1.1 物质的分类

---

## 一、概念

### 1. 为什么要分类？

化学研究对象极其庞杂，**分类**是为了：

- 发现规律（同类物质性质相似）
- 预测性质（由类别推断可能反应）
- 简化记忆（树状结构一目了然）

### 2. 两种分类方法

| 方法 | 含义 | 特点 |
|------|------|------|
| **交叉分类法** | 从不同角度对同一事物分类 | 同一物质可属多个「类别」 |
| **树状分类法** | 按层次逐步细分 | 形成「物质分类树」，不重不漏 |

**交叉分类 vs 树状分类：**

```
交叉分类：Na₂CO₃ 既是「盐」，又可按阳离子分为「钠盐」、按阴离子分为「碳酸盐」
树状分类：物质 → 纯净物 → 化合物 → 盐 → …
```

### 3. 物质的基本分类（树状）

```
物质
├── 纯净物（一种物质）
│   ├── 单质（同种元素）
│   │   ├── 金属单质：Fe、Cu、Na…
│   │   └── 非金属单质：O₂、Cl₂、S、C…
│   └── 化合物（不同种元素）
│       ├── 氧化物：CO₂、Fe₂O₃、H₂O…
│       ├── 酸：HCl、H₂SO₄、HNO₃…
│       ├── 碱：NaOH、Ca(OH)₂…
│       └── 盐：NaCl、Na₂CO₃、CuSO₄…
└── 混合物（多种物质，无固定比例）
    ├── 溶液、胶体、浊液（分散系）
    ├── 空气、合金、石油…
```

**核心判断：**

| 概念 | 定义 | 关键 |
|------|------|------|
| **纯净物** | 只含一种物质 | 有固定组成，有固定熔沸点（一般） |
| **混合物** | 含两种及以上物质 | 无固定组成，各成分保持原有性质 |
| **单质** | 同种元素组成的纯净物 | O₂ 是单质，O₃ 也是单质（同素异形体） |
| **化合物** | 不同种元素组成的纯净物 | 元素以固定比例结合 |

### 4. 氧化物、酸、碱、盐的判定

| 类别 | 组成特点 | 判断要点 |
|------|----------|----------|
| **氧化物** | 两种元素，其中一种是 O | 如 CO₂、Fe₂O₃；H₂O 也是氧化物 |
| **酸** | 电离出的阳离子**全部是 H⁺** | HCl、H₂SO₄、HNO₃ |
| **碱** | 电离出的阴离子**全部是 OH⁻** | NaOH、Ca(OH)₂ |
| **盐** | 电离出的阳离子无 H⁺、阴离子无 OH⁻ | NaCl、Na₂CO₃、CuSO₄ |

**特殊物质：**

- **Na₂CO₃、NaHCO₃**：是**盐**，不是碱（Na₂CO₃ 溶液呈碱性，因 CO₃²⁻ 水解）
- **Cu(OH)₂**：是**碱**（难溶碱仍是碱）

### 5. 分散系（延伸）

当一种物质分散到另一种物质中，形成**分散系**（属于混合物）：

| 术语 | 含义 |
|------|------|
| **分散质** | 被分散的物质 |
| **分散剂** | 起分散作用的物质（通常是水） |

**按分散质粒子直径分类：**

| 类型 | 粒子直径 | 外观 | 透过滤纸 | 透过半透膜 |
|------|----------|------|----------|------------|
| **溶液** | < 1 nm | 均一透明 | 能 | 能 |
| **胶体** | 1～100 nm | 均一，可能有颜色 | 能 | **不能** |
| **浊液** | > 100 nm | 不均一 | 不能 | 不能 |

**丁达尔效应：** 可见光通过胶体时，光路可见（光散射），可用于鉴别胶体与溶液。

详见 [[concepts/分散系]]。

## 二、举例

### 1. 交叉分类法

**以 Na₂CO₃ 为例：**

| 分类角度 | 所属类别 |
|----------|----------|
| 组成 | 化合物 → 盐 |
| 阳离子 | 钠盐 |
| 阴离子 | 碳酸盐 |
| 溶解性 | 易溶盐 |

### 2. 树状分类

| 物质 | 分类路径 |
|------|----------|
| 铁 Fe | 物质 → 纯净物 → 单质 → 金属单质 |
| 氧气 O₂ | 物质 → 纯净物 → 单质 → 非金属单质 |
| 盐酸 HCl（水溶液） | 混合物（溶液）；溶质 HCl 属于酸 |
| 氯化钠 NaCl | 物质 → 纯净物 → 化合物 → 盐 |
| 干冰 CO₂ | 物质 → 纯净物 → 化合物 → 氧化物 |
| 空气 | 物质 → 混合物 |
| 生铁 | 物质 → 混合物（合金） |

### 3. 分散系举例

| 分散系 | 分散质 | 分散剂 | 实例 |
|--------|--------|--------|------|
| 溶液 | Na⁺、Cl⁻ | H₂O | 食盐水 |
| 溶液 | 蔗糖分子 | H₂O | 蔗糖水 |
| 胶体 | Fe(OH)₃ 胶粒 | H₂O | 氢氧化铁胶体 |
| 胶体 | 蛋白质分子 | H₂O | 鸡蛋清溶于水 |
| 胶体 | 淀粉分子 | H₂O | 淀粉溶液（实为胶体） |
| 浊液 | 泥土颗粒 | H₂O | 泥浆 |
| 浊液 | 小油滴 | H₂O | 乳浊液（牛奶） |

### 4. 氢氧化铁胶体制备（必考实验）

```
FeCl₃ + 3H₂O（沸水）→ Fe(OH)₃（胶体）+ 3HCl
```

操作：向沸水中滴加饱和 FeCl₃ 溶液，至液体呈**红褐色**。

现象：激光照射出现光路（丁达尔效应）。

## 三、拓展

### 1. 同素异形体

同种元素形成的**不同单质**，如 O₂ 与 O₃、金刚石与石墨。互为同素异形体，**不是**同分异构体。

### 2. 溶液 vs 胶体 vs 浊液

分类依据是**分散质粒子直径**，不是「看起来清不清」：

- 淀粉溶液、蛋白质溶液：看起来「清」，实为**胶体**
- 纳米材料（1～100 nm）：按粒径可归为胶体范畴

### 3. 胶体的应用

| 应用 | 原理 |
|------|------|
| 明矾净水 | Al(OH)₃ 胶体吸附悬浮杂质 |
| 血液透析 | 半透膜：小分子通过，蛋白质（胶体粒子）不能通过 |
| 豆浆/牛奶 | 胶体体系 |

### 4. 渗析

用半透膜分离胶体与溶液中离子：Fe(OH)₃ 胶体中的 Cl⁻ 可透过半透膜，Fe(OH)₃ 胶粒不能 → 提纯胶体。

| 本节内容 | 后续用途 |
|----------|----------|
| 酸/碱/盐分类 | 第 2 章离子反应、电离方程式 |
| 溶液 | 物质的量浓度 c = n/V |

## 四、易错点

### 纯净物 vs 混合物

| 说法 | 正误 | 解析 |
|------|------|------|
| 冰水混合物是纯净物 | ✅ | 只有 H₂O 一种物质 |
| 洁净的空气是纯净物 | ❌ | 多种气体组成的混合物 |
| 盐酸是纯净物 | ❌ | HCl 的水溶液，是混合物 |
| 干冰是纯净物 | ✅ | 只有 CO₂ |
| 漂白粉是纯净物 | ❌ | Ca(ClO)₂ 与 CaCl₂ 等的混合物 |

### 单质 vs 化合物

| 说法 | 正误 | 解析 |
|------|------|------|
| 含同种元素的物质一定是单质 | ❌ | O₂ 与 O₃ 混合仍是混合物 |
| 由两种元素组成的物质一定是化合物 | ❌ | 可能是混合物（如 H₂ 与 O₂ 混合） |

### 酸/碱/盐

| 说法 | 正误 | 解析 |
|------|------|------|
| Na₂CO₃ 是碱 | ❌ | 是盐；溶液碱性来自 CO₃²⁻ 水解 |
| Cu(OH)₂ 不是碱 | ❌ | 难溶，但仍是碱 |
| H₂O 是氧化物 | ✅ | 两种元素，其中一种是 O |

### 分散系（高频）

| 说法 | 正误 | 解析 |
|------|------|------|
| 胶体是纯净物 | ❌ | 胶体是**分散系**，属于混合物 |
| 胶体有丁达尔效应，溶液没有 | ✅ | 鉴别胶体与溶液的经典方法 |
| 能通过滤纸的一定是溶液 | ❌ | 胶体也能透过滤纸 |
| 能通过半透膜的一定是溶液 | ✅ | 胶体粒子不能透过半透膜 |
| 淀粉溶液是溶液 | ❌ | 淀粉分子直径在胶体范围，是**胶体** |
| 1 mol Fe³⁺ 水解得到 1 mol Fe(OH)₃ 胶体粒子 | ❌ | 胶体粒子是多个 Fe(OH)₃ 聚集，粒子数远小于 N_A |

## 五、速记卡片

```
物质 ─┬─ 纯净物 ─┬─ 单质（金属/非金属）
      │          └─ 化合物（氧化物/酸/碱/盐）
      └─ 混合物 ─┬─ 分散系 ─┬─ 溶液（<1 nm）
                 │          ├─ 胶体（1~100 nm）→ 丁达尔效应
                 │          └─ 浊液（>100 nm）
                 └─ 空气、合金、石油…

鉴别胶体 vs 溶液：丁达尔效应
鉴别胶体 vs 浊液：外观均一性 / 能否透过滤纸
```
