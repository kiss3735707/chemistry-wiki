# 物质的量

## 定义

- **物质的量** n：表示含有一定数目粒子的集合体，单位 **mol**
- **阿伏加德罗常数** N_A ≈ 6.02×10²³ mol⁻¹
- **摩尔质量** M：单位物质的量的物质质量，g/mol；数值上等于相对原子/分子质量
- **气体摩尔体积** V_m：标况下约 **22.4 L/mol**（仅适用于气体）
- **物质的量浓度** c = n/V（V 为溶液体积，L）

## 要点

**换算关系：**

```
n = m/M = N/N_A = V/V_m（标况气体）
c = n/V；稀释：c₁V₁ = c₂V₂
```

| 物理量 | 单位 |
|--------|------|
| n | mol |
| m | g |
| V（气体，标况） | L |
| c | mol/L |

## 易错点

- 22.4 L/mol **不适用于**液体、固体
- 「1 mol 物质」与「1 mol 粒子」要区分
- 配制溶液时，仰视定容 → c **偏小**；俯视 → c **偏大**

## 例题/应用

标况下 11.2 L CO₂ 的物质的量：n = 11.2/22.4 = **0.5 mol**
