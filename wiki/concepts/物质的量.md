# 物质的量

> 沪科版高一必修第一册 · 第 1 章 化学研究的天地 · 1.2 物质的量

---

## 一、概念

### 1. 为什么引入物质的量？

宏观物质用**质量**（g）衡量，微观粒子用**个数**衡量，两者数量级差距极大（1 滴水 ≈ 10²¹ 个水分子）。**物质的量**是连接宏观与微观的桥梁。

### 2. 基本物理量一览

| 物理量 | 符号 | 单位 | 含义 |
|--------|------|------|------|
| **物质的量** | n | mol（摩尔） | 含有一定数目粒子的集合体 |
| **阿伏加德罗常数** | N_A | mol⁻¹ | 1 mol 粒子所含的粒子数 |
| **摩尔质量** | M | g/mol | 单位物质的量的物质质量 |
| **气体摩尔体积** | V_m | L/mol | 单位物质的量的气体体积 |
| **物质的量浓度** | c | mol/L | 单位体积溶液中溶质的物质的量 |

### 3. 物质的量与阿伏加德罗常数

- **定义**：物质的量表示含有一定数目粒子的集合体，单位是 **mol**
- **阿伏加德罗常数** N_A ≈ **6.02×10²³ mol⁻¹**
- **含义**：1 mol 任何粒子（原子、分子、离子、电子等）所含的粒子数相同

```
n = N / N_A
```

**规范表述示例：**

- 0.2 mol H₂ 中含氢原子 **0.4 mol**（或 0.4×N_A 个）
- 6.02×10²² 个 Na⁺ 的物质的量为 **0.1 mol**

### 4. 摩尔质量

- **定义**：单位物质的量的物质所具有的质量，M = m/n
- **单位**：g/mol
- **重要规律**：摩尔质量在**数值上**等于相对原子质量或相对分子质量

| 物质 | 相对分子/原子质量 | 摩尔质量 M |
|------|-------------------|------------|
| Fe | 56 | 56 g/mol |
| H₂O | 18 | 18 g/mol |
| CO₂ | 44 | 44 g/mol |
| NaCl | 58.5 | 58.5 g/mol |

```
n = m / M
```

### 5. 气体摩尔体积

- **定义**：单位物质的量的气体在**一定温度和压强下**所占的体积
- **标准状况（标况）**：0 ℃、101 kPa（或 101.325 kPa）
- **标况下**：V_m ≈ **22.4 L/mol**（仅适用于**气体**）

```
n = V / V_m        （标况气体）
V = n × 22.4 L     （标况气体）
```

**影响气体体积的因素：** 粒子数目、粒子间距（温度、压强）。同温同压下，相同物质的量的不同气体，体积**相同**。

### 6. 物质的量浓度

- **定义**：单位体积溶液中所含溶质的物质的量

```
c = n / V
```

- **V** 是**溶液体积**（L），不是溶剂体积
- **稀释定律**：稀释前后溶质物质的量不变 → **c₁V₁ = c₂V₂**

### 7. 换算关系总图

```
                    ┌──×M──→ 质量 m（g）
                    │         │
                    │         └──÷M──┐
                    │                 ↓
粒子数 N ──÷N_A──→ 物质的量 n ──×N_A──→ 粒子数 N
                    │                 ↑
                    │         ┌──×V_m──┘
                    └──÷V_m──→ 气体体积 V（L，标况）
                    │
                    └──÷V──→ 物质的量浓度 c（mol/L）
```

## 二、举例

### 1. 物质的量 ↔ 粒子数

**例 1**：0.5 mol H₂SO₄ 中含多少 mol 的 H、S、O 原子？

| 粒子 | 计算 | 结果 |
|------|------|------|
| H | 0.5 × 2 | 1 mol |
| S | 0.5 × 1 | 0.5 mol |
| O | 0.5 × 4 | 2 mol |

**例 2**：标准状况下 33.6 L CO₂ 的物质的量？

```
n = 33.6 / 22.4 = 1.5 mol
```

### 2. 物质的量 ↔ 质量

**例 3**：88 g CO₂ 的物质的量？

```
n = 88 / 44 = 2 mol
```

**例 4**：0.25 mol NaCl 的质量？

```
m = 0.25 × 58.5 = 14.625 g
```

### 3. 混合气体（同温同压）

**例 5**：标况下 11.2 L H₂ 和 22.4 L O₂ 混合，总体积？

```
n(H₂) = 11.2/22.4 = 0.5 mol
n(O₂) = 22.4/22.4 = 1 mol
n(总) = 1.5 mol → V(总) = 1.5 × 22.4 = 33.6 L
```

**规律**：同温同压下 V₁/V₂ = n₁/n₂

### 4. 物质的量浓度

**例 6**：将 20 mL 5 mol/L 的 HCl 稀释至 500 mL，求稀释后浓度。

```
c₁V₁ = c₂V₂
5 × 0.02 = c₂ × 0.5
c₂ = 0.2 mol/L
```

**例 7**：配制 250 mL 0.1 mol/L NaOH 溶液，需 NaOH 固体多少克？

```
n = c × V = 0.1 × 0.25 = 0.025 mol
m = 0.025 × 40 = 1.0 g
```

### 5. 综合计算（化学方程式）

**例 8**：标况下 5.6 g 铁与足量稀 H₂SO₄ 反应，生成 H₂ 的体积？

```
Fe + H₂SO₄ → FeSO₄ + H₂↑
n(Fe) = 5.6/56 = 0.1 mol
n(H₂) = 0.1 mol
V(H₂) = 0.1 × 22.4 = 2.24 L
```

### 6. 测定气体摩尔体积（活动/实验）

**原理**：用已知质量的金属（如 Mg）与酸反应产生 H₂，测得气体体积，计算 V_m。

```
Mg + 2HCl → MgCl₂ + H₂↑
V_m = V(H₂) / n(Mg)
```

**要点**：检查装置气密性；冷却至室温再读体积；注意收集气体是否含水蒸气。

## 三、拓展

### 1. 摩尔质量的其他求法

| 已知条件 | 公式 |
|----------|------|
| 定义式 | M = m/n |
| 已知分子质量 m(分子) | M = m(分子) × N_A |
| 标况气体密度 ρ（g/L） | M = 22.4 × ρ |
| 同温同压相对密度 D | M(A) = M(B) × D |
| 混合气体平均摩尔质量 | M̄ = M₁a% + M₂b% + … |

### 2. 阿伏加德罗常数的考查角度

- 1 mol 物质**质量**是多少（g）
- 1 mol 物质含**多少 mol 原子/分子**（看化学式）
- 1 mol 物质含**多少个粒子**（×N_A）
- **注意**：1 mol 某物质 ≠ 1 mol 某粒子（如 1 mol H₂O 含 3 mol 原子）

### 3. 标况 vs 非标况

| 情况 | 能否用 22.4 L/mol |
|------|-------------------|
| 标况下的 O₂、H₂、CO₂ 等气体 | ✅ |
| 标况下的 H₂O、乙醇（液体） | ❌ |
| 25 ℃ 101 kPa 下的气体 | ❌（需用理想气体状态方程，高一一般不要求） |

### 4. 溶液浓度两种表示的换算

| 表示法 | 公式 | 适用 |
|--------|------|------|
| 溶质质量分数 ω | ω = m(溶质)/m(溶液) × 100% | 初中 |
| 物质的量浓度 c | c = n/V | 高中 |

稀溶液近似：ρ ≈ 1 g/mL 时，ω 与 c 可互算（需知道 M）。

### 5. 与后续章节的联系

| 本节 | 后续应用 |
|------|----------|
| n = m/M | 化学方程式计算、纯度测定 |
| c = n/V | 1.3 配制一定物质的量浓度的溶液 |
| 气体体积换算 | 第 2 章 Cl₂、H₂ 制取与收集 |
| 粒子数计算 | 第 4 章原子结构、阿伏加德罗常数微观含义 |

### 6. 与初中的衔接

初中用**溶质质量分数** ω 和 m₁ω₁ = m₂ω₂；高一引入 **c** 和 c₁V₁ = c₂V₂。量筒读数误差规律相同：仰视 → 浓度偏小，俯视 → 浓度偏大。详见 [[concepts/溶液与溶质质量分数]]。

## 四、易错点

### 概念辨析

| 说法 | 正误 | 解析 |
|------|------|------|
| 摩尔是物质的质量单位 | ❌ | 摩尔是**物质的量**的单位 |
| 1 mol 任何物质的质量都相同 | ❌ | 1 mol 物质**质量**不同（M 不同） |
| 1 mol 任何物质的粒子数相同 | ✅ | 粒子**数目**都是 N_A 个 |
| 22.4 L 任何气体在标况下都是 1 mol | ✅ | 标况气体适用 |
| 22.4 L H₂O 在标况下是 1 mol | ❌ | 标况下水是**液体** |
| 64 g O₂ 含 2 mol 氧原子 | ❌ | 64 g O₂ = 2 mol O₂ 分子 = **4 mol** 氧原子 |

### 化学式中的粒子关系

| 物质 | 1 mol 该物质含… |
|------|-----------------|
| H₂ | 1 mol 分子，**2 mol** H 原子 |
| H₂SO₄ | 1 mol 分子，2 mol H、1 mol S、4 mol O 原子 |
| NaCl | 1 mol 晶体，1 mol Na⁺ + 1 mol Cl⁻（无 NaCl 分子） |

### 浓度与稀释

| 说法 | 正误 | 解析 |
|------|------|------|
| c = n/V 中 V 是溶剂体积 | ❌ | V 是**溶液体积** |
| 将 1 mol NaCl 溶于 1 L 水，c = 1 mol/L | ❌ | 溶液体积 > 1 L，c < 1 mol/L |
| 稀释时溶质质量不变 | ✅ | 稀释前后 n(溶质) 不变 |

### 配制溶液误差（与 1.3 衔接）

| 操作 | 对 c 的影响 |
|------|-------------|
| 定容时**仰视**（凹液面读数偏小） | c **偏小** |
| 定容时**俯视** | c **偏大** |
| 未洗涤烧杯和玻璃棒 | c **偏小**（溶质损失） |
| 容量瓶未干燥（内壁有水） | **无影响**（定容时会稀释） |

### N_A 相关陷阱

| 说法 | 正误 | 解析 |
|------|------|------|
| 1 mol Fe³⁺ 完全水解得 N_A 个 Fe(OH)₃ 胶体粒子 | ❌ | 胶体粒子是**聚集体**，数目远小于 N_A |
| 标况下 22.4 L 混合气体含 N_A 个分子 | ✅ | 1 mol 气体分子 |
| 28 g N₂ 和 CO 混合气体，分子数相同 | ✅ | 两者 M 均为 28 g/mol，n 相同则 N 相同 |

## 五、速记卡片

```
核心公式：
  n = m/M = N/N_A = V/22.4（标况气体）
  c = n/V；稀释 c₁V₁ = c₂V₂

标况：0 ℃、101 kPa；V_m ≈ 22.4 L/mol（仅气体！）

1 mol 粒子数 = N_A ≈ 6.02×10²³
M 数值 = 相对原子/分子质量

解题步骤：找关系式 → 统一单位（L、mL 换算）→ 列比例 → 检查过量
```
