# 化学实验方法

> 沪科版高一必修第一册 · 第 1 章 化学研究的天地 · 1.3 化学中常用的实验方法

---

## 一、概念

### 1. 本节四大模块

| 模块 | 核心问题 |
|------|----------|
| **物质的制备** | 选什么装置、怎么收集气体 |
| **分离与提纯** | 混合物各组分性质不同，如何分开 |
| **物质的检验** | 如何确认某种离子或物质存在 |
| **溶液配制** | 如何准确配制一定物质的量浓度的溶液 |

### 2. 物质的制备——气体发生装置

根据反应物状态和反应条件选择：

| 类型 | 条件 | 典型反应 | 装置特点 |
|------|------|----------|----------|
| **固 + 固加热** | 需加热 | 2KClO₃ → 2KCl + 3O₂↑ | 试管口略向下倾斜 |
| **固 + 液不加热** | 常温 | Zn + H₂SO₄ → ZnSO₄ + H₂↑ | 长颈漏斗或分液漏斗 |
| **固 + 液加热** | 需加热 | MnO₂ + 4HCl → MnCl₂ + Cl₂↑ + 2H₂O | 需尾气吸收（有毒气体） |

**气体收集方法：**

| 方法 | 适用气体 | 依据 |
|------|----------|------|
| **向上排空气法** | 密度 > 空气（如 CO₂、Cl₂） | 气体从集气瓶底部向上置换空气 |
| **向下排空气法** | 密度 < 空气（如 H₂、NH₃） | 气体从顶部向下置换空气 |
| **排水法** | 难溶于水且不与水反应（如 O₂、H₂、CO） | 利用水将气体排出 |

**气密性检查：** 将导管一端浸入水中，双手捂热容器（或加热），导管口冒气泡 → 松开手后形成一段水柱 → 气密性良好。

### 3. 分离与提纯——常用方法

| 方法 | 原理 | 适用实例 | 注意事项 |
|------|------|----------|----------|
| **过滤** | 颗粒大小不同 | 分离不溶性固体与液体（泥沙水） | 一贴二低三靠 |
| **蒸发** | 溶剂挥发 | 从溶液中获得可溶性固体（NaCl） | 接近干时停止加热，利用余热蒸干 |
| **蒸馏** | 沸点不同 | 分离互溶液体（酒精与水） | 温度计水银球位于支管口处 |
| **分液** | 互不相溶 | 分离分层液体（油与水、CCl₄ 与水） | 分液漏斗下端管口紧贴烧杯内壁 |
| **萃取** | 溶质在两种溶剂中溶解度不同 | 用 CCl₄ 从碘水中萃取 I₂ | 萃取后通常需分液 |
| **重结晶** | 溶解度随温度变化大 | 提纯 KNO₃（从 KNO₃ 和 NaCl 混合物） | 趁热过滤，冷却结晶 |

**方法选择思路：**

```
混合物
├── 固 + 液（不溶）→ 过滤
├── 固 + 液（可溶，要固体）→ 蒸发 / 重结晶
├── 两种互溶液体（沸点差大）→ 蒸馏
├── 两种互溶液体（沸点接近，萃取剂选择性好）→ 萃取 + 分液
└── 两种互不相溶液体 → 分液
```

### 4. 物质的检验

**检验原则：**

- 试剂特征性（只与待检离子反应产生明显现象）
- 先加酸排除干扰（如检验 SO₄²⁻ 前先排除 CO₃²⁻）
- 检验 CO₃²⁻ 用酸，检验 SO₄²⁻ 用 Ba²⁺，**顺序不能颠倒**（先检验 CO₃²⁻）

**常见阳离子检验（高一）：**

| 离子 | 试剂 | 现象 |
|------|------|------|
| NH₄⁺ | NaOH 溶液，加热 | 产生使湿润红色石蕊试纸变蓝的 gas |
| Fe³⁺ | NaOH 溶液 | 红褐色沉淀 |
| Ba²⁺ | 稀 H₂SO₄ 或可溶性硫酸盐 | 白色沉淀（BaSO₄） |

**常见阴离子检验：**

| 离子 | 试剂 | 现象 | 注意 |
|------|------|------|------|
| Cl⁻ | AgNO₃ + 稀 HNO₃ | 白色沉淀 AgCl | HNO₃ 排除 CO₃²⁻ 等干扰 |
| SO₄²⁻ | BaCl₂ + 稀 HCl | 白色沉淀 BaSO₄ | 先酸化排除 CO₃²⁻ |
| CO₃²⁻ | 稀盐酸 | 产生使澄清石灰水变浑浊的气体 | 先检验 CO₃²⁻ |
| HCO₃⁻ | 稀盐酸 | 产生 CO₂（现象同 CO₃²⁻） | 需结合其他信息区分 |

更完整的初中检验表见 [[concepts/物质鉴别]]。

### 5. 配制一定物质的量浓度的溶液

**所需仪器：** 天平（或量筒）、烧杯、玻璃棒、容量瓶、胶头滴管

**操作步骤：**

```
计算 → 称量（或量取）→ 溶解 → 转移 → 洗涤 → 定容 → 摇匀 → 贴标签
```

| 步骤 | 要点 |
|------|------|
| **计算** | n = c×V；m = n×M |
| **称量/量取** | 固体用天平；液体用量筒或移液管 |
| **溶解** | 在烧杯中溶解，**不能**在容量瓶中直接溶解固体 |
| **转移** | 冷却至室温后，沿玻璃棒注入容量瓶 |
| **洗涤** | 洗涤烧杯和玻璃棒 2～3 次，洗涤液一并转入（**溶质不损失**） |
| **定容** | 加蒸馏水至距刻度线 1～2 cm，改用胶头滴管滴加至**凹液面最低处与刻度线相切** |
| **摇匀** | 塞紧瓶塞，反复上下颠倒摇匀 |

**容量瓶使用：** 只能配制溶液、量取体积，**不能**用来溶解固体、加热、长期储存。

## 二、举例

### 1. 气体制备与收集

**例 1**：实验室制 H₂

```
Zn + H₂SO₄ → ZnSO₄ + H₂↑
```

- 装置：固 + 液，不加热
- 收集：向下排空气法 或 排水法
- 验纯：点燃，听到尖锐爆鸣声说明不纯

**例 2**：实验室制 CO₂

```
CaCO₃ + 2HCl → CaCl₂ + H₂O + CO₂↑
```

- 装置：固 + 液，不加热
- 收集：向上排空气法
- 检验：通入澄清石灰水，变浑浊

**例 3**：制 Cl₂（第 2 章预习）

```
MnO₂ + 4HCl(浓) → MnCl₂ + Cl₂↑ + 2H₂O
```

- 装置：固 + 液，加热
- 收集：向上排空气法
- **尾气吸收**：NaOH 溶液（Cl₂ 有毒）

### 2. 分离提纯实例

**例 4**：从碘水中提取碘

- 方法：**萃取 + 分液**
- 萃取剂：CCl₄ 或苯（碘在有机溶剂中溶解度更大）
- 操作：振荡 → 静置分层 → 分液漏斗取下层（CCl₄ 层，紫色）

**例 5**：粗盐提纯（溶解 → 过滤 → 蒸发）

```
泥沙水 → 过滤 → 澄清 NaCl 溶液 → 蒸发 → NaCl 固体
```

**例 6**：酒精与水的分离

- 方法：**蒸馏**（沸点：乙醇 78 ℃，水 100 ℃）

### 3. 离子检验实例

**例 7**：检验某溶液是否含 SO₄²⁻

1. 取少量溶液于试管
2. 先加稀 HCl 酸化（排除 CO₃²⁻）
3. 再加 BaCl₂ 溶液
4. 产生白色沉淀 → 含 SO₄²⁻

**例 8**：检验 NH₄⁺

```
NH₄⁺ + OH⁻ → NH₃↑ + H₂O（加热）
```

湿润红色石蕊试纸变蓝。

### 4. 配制溶液实例

**例 9**：配制 500 mL 0.1 mol/L NaOH 溶液

```
n = 0.1 × 0.5 = 0.05 mol
m = 0.05 × 40 = 2.0 g
```

称 2.0 g NaOH → 烧杯溶解 → 冷却 → 转入 500 mL 容量瓶 → 洗涤 → 定容 → 摇匀。

**注意**：NaOH 易潮解、有腐蚀性，用烧杯称量（或快速称量），溶解时放热，**必须冷却**后再转移。

## 三、拓展

### 1. 萃取剂选择原则

- 与原溶剂**互不相溶**
- 对溶质的溶解度**远大于**原溶剂
- 不与溶质、原溶剂反应
- 常见：CCl₄、苯（萃取 Br₂、I₂）；乙酸乙酯等

### 2. 蒸馏 vs 分馏

| | 蒸馏 | 分馏 |
|---|------|------|
| 适用 | 沸点相差较大（>30 ℃） | 沸点相差较小 |
| 装置 | 蒸馏烧瓶 + 直形冷凝管 | 分馏柱 |
| 实例 | 酒精与水 | 石油分馏 |

### 3. 重结晶提纯步骤

1. 加热溶解（配成热饱和溶液）
2. **趁热过滤**（除去不溶性杂质）
3. 冷却结晶
4. 过滤、洗涤、干燥

### 4. 实验安全要点

| 情况 | 处理 |
|------|------|
| 浓酸/浓碱沾皮肤 | 大量水冲洗，碱用硼酸，酸用 NaHCO₃ |
| 少量 Cl₂ 泄漏 | 用 NaOH 溶液或 Na₂S₂O₃ 吸收 |
| 加热试管 | 试管口不对人，液体不超过 1/3 |
| 稀释浓硫酸 | **酸入水**，沿器壁缓慢加入，搅拌 |

## 四、易错点

### 装置与操作

| 说法 | 正误 | 解析 |
|------|------|------|
| 在容量瓶中溶解 NaCl 固体 | ❌ | 应在**烧杯**中溶解后转移 |
| 定容时俯视刻度线 | ❌ | 加水偏少，c **偏大** |
| 定容时仰视刻度线 | ❌ | 加水偏多，c **偏小** |
| 转移后未洗涤烧杯 | ❌ | 溶质损失，c **偏小** |
| 容量瓶未干燥（内壁有水） | ✅ 无影响 | 定容时会稀释至刻度，n 不变 |
| 溶解后立即定容（NaOH 未冷却） | ❌ | 热胀冷缩，冷却后 c **偏大** |
| 蒸馏时温度计插入液面以下 | ❌ | 应测**蒸气**温度，水银球在支管口 |

### 分离方法选择

| 说法 | 正误 | 解析 |
|------|------|------|
| 分离碘和食盐固体用萃取 | ❌ | 两者均为固体，应**溶解后**萃取或升华 |
| 分离 Fe 粉和 Cu 粉用过滤 | ❌ | 两者均不溶于水，用**磁铁**或浮选等物理方法 |
| 分离油和水的混合物用分液 | ✅ | 互不相溶，直接分液 |

### 离子检验

| 说法 | 正误 | 解析 |
|------|------|------|
| 检验 Cl⁻ 用 AgNO₃ 即可 | ❌ | 需加稀 HNO₃ 排除 CO₃²⁻ 等 |
| 先加 BaCl₂ 再酸化检验 SO₄²⁻ | ❌ | 应先**酸化**再加 Ba²⁺ |
| AgCl 白色沉淀加稀硝酸溶解 | ❌ | AgCl **不溶于**稀硝酸 |

### 配制误差汇总

| 操作 | 对 c 的影响 |
|------|-------------|
| 称量时砝码生锈（实际质量偏大） | c **偏大** |
| 称量时物码颠倒（使用游码时） | c **偏小** |
| 定容后未摇匀 | 不影响 c（均匀后仍准确） |
| 读数时仰视（量筒量取浓溶液） | 量取体积偏大，c **偏大** |

## 五、速记卡片

```
气体制备三类型：固固加热 / 固液不加热 / 固液加热
气体收集三方法：上排空气 / 下排空气 / 排水法

分离提纯六方法：过滤、蒸发、蒸馏、分液、萃取、重结晶

配制溶液：算→称→溶→转→洗→定→摇
定容口诀：凹液面最低处与刻度线相切

检验顺序：CO₃²⁻（加酸）→ SO₄²⁻（酸化 + Ba²⁺）→ Cl⁻（HNO₃ + Ag⁺）
```
