## 2.1 海水中的氯

### 一、概念

#### 1. 粗盐提纯

粗盐含 **NaCl** 及可溶性杂质（Ca²⁺、Mg²⁺、SO₄²⁻ 等）和不溶性泥沙。

**除杂原理与试剂：**

| 杂质离子 | 除杂试剂 | 生成沉淀 |
|----------|----------|----------|
| SO₄²⁻ | BaCl₂ | BaSO₄↓ |
| Ca²⁺、过量 Ba²⁺ | Na₂CO₃ | CaCO₃↓、BaCO₃↓ |
| Mg²⁺ | NaOH | Mg(OH)₂↓ |
| 过量 CO₃²⁻、OH⁻ | 稀 HCl | 转化为气体或 H₂O |

**操作顺序（必背）：**

```
溶解 → 过滤（除泥沙）→ 加 BaCl₂ → 加 Na₂CO₃ → 加 NaOH
    → 过滤 → 加稀 HCl 至无气泡 → 蒸发
```

**顺序关键：** BaCl₂ 必须在 Na₂CO₃ **之前**（先除 SO₄²⁻，再用 CO₃²⁻ 除去过量 Ba²⁺）。

#### 2. 电解饱和食盐水

```
2NaCl + 2H₂O ──电解──→ 2NaOH + Cl₂↑ + H₂↑
```

| 电极 | 反应 | 产物 |
|------|------|------|
| **阳极** | 2Cl⁻ → Cl₂↑ + 2e⁻ | Cl₂ |
| **阴极** | 2H₂O + 2e⁻ → H₂↑ + 2OH⁻ | H₂、NaOH |

**工业意义：** 氯碱工业——同时获得 Cl₂、H₂、NaOH 三种重要化工原料。

#### 3. 氯气

详见 [[entities/氯气]]。

| 项目 | 内容 |
|------|------|
| 物理性质 | 黄绿色、有毒、有刺激性；密度 > 空气 |
| 化学性质 | **强氧化性**（活泼性 Cl > Br > I） |
| 主要反应 | 与 Fe、Cu、H₂、H₂O、NaOH 等 |

#### 4. 次氯酸与次氯酸盐

| 物质 | 要点 |
|------|------|
| **HClO** | 弱酸、**不稳定**（见光分解）；有漂白性、消毒性 |
| **NaClO** | 漂白液有效成分；Cl₂ + 2NaOH → NaCl + NaClO + H₂O |
| **Ca(ClO)₂** | 漂白粉有效成分；Ca(OH)₂ + Cl₂ → Ca(ClO)₂ + … |
| **漂白原理** | Ca(ClO)₂ + CO₂ + H₂O → CaCO₃ + 2HClO（酸化生成 HClO） |

---

### 二、举例

**例 1**：粗盐中 SO₄²⁻ 检验——先加 BaCl₂ 产生白色沉淀。

**例 2**：Cl₂ 与 Fe 反应

```
2Fe + 3Cl₂ → 2FeCl₃    （点燃或加热，生成棕黄色烟）
```

**例 3**：Cl₂ 与 NaOH（冷、稀）

```
Cl₂ + 2NaOH → NaCl + NaClO + H₂O
```

Cl 从 0 价 → -1（NaCl）和 +1（NaClO），**歧化反应**。

**例 4**：氯水漂白

```
Cl₂ + H₂O ⇌ HCl + HClO
```

HClO 具有强氧化性，使有色物质褪色（不可逆，见 [[comparisons/SO₂漂白 vs HClO漂白]]）。

---

### 三、拓展

- **氯碱工业**：电解饱和食盐水是化学工业基础流程之一
- **84 消毒液**：含 NaClO，不能与洁厕灵（HCl）混用（产生有毒 Cl₂）
- **HClO 分解**：2HClO → 2HCl + O₂↑（见光分解，需避光保存）

---

### 四、易错点（2.1）

| 说法 | 正误 | 解析 |
|------|------|------|
| 先加 Na₂CO₃ 再加 BaCl₂ | ❌ | Ba²⁺ 无法除去 |
| 除杂后不加 HCl 直接蒸发 | ❌ | 残留 OH⁻、CO₃²⁻ 影响纯度 |
| Cl₂ 使干燥有色布条褪色 | ❌ | 需**湿润**布条（Cl₂ + H₂O → HClO） |
| 漂白粉有效成分为 CaCl₂ | ❌ | 是 **Ca(ClO)₂** |

## 2.2 氧化还原反应和离子反应

> 本章系统学习两种核心反应类型。完整条目见 [[concepts/氧化反应与还原反应]]、[[concepts/离子反应]]。

### 一、概念

#### 1. 氧化还原反应

**本质：** 有**电子转移**（或化合价升降）的反应。

| 角色 | 变化 | 口诀 |
|------|------|------|
| **氧化剂** | 得电子，化合价降低 | 降得氧（被还原） |
| **还原剂** | 失电子，化合价升高 | 升失还（被氧化） |

**本章典型氧化剂：** Cl₂、Br₂、O₂

**本章典型反应：**

```
Cl₂ + 2Br⁻ → Br₂ + 2Cl⁻    （Cl₂ 氧化 Br⁻）
Cl₂ + 2I⁻ → I₂ + 2Cl⁻      （Cl₂ 氧化 I⁻）
```

#### 2. 电解质与电离

| 类型 | 特点 | 实例 |
|------|------|------|
| **强电解质** | 水溶液中**完全**电离 | HCl、NaOH、NaCl、BaCl₂ |
| **弱电解质** | **部分**电离 | HClO、NH₃·H₂O、H₂CO₃ |

#### 3. 离子反应

**发生条件（满足其一）：**

- 生成沉淀
- 生成气体
- 生成弱电解质（H₂O、HClO…）
- **氧化还原型**（如 Cl₂ + 2I⁻ → I₂ + 2Cl⁻）

**离子方程式四步：** 写 → 改 → 删 → 查（原子守恒 + **电荷守恒**）

---

### 二、举例

**例 5**：判断 Cl₂ + 2NaOH → NaCl + NaClO + H₂O 中的氧化剂、还原剂

- Cl₂ 中 Cl 为 0 价 → NaCl 中 -1、NaClO 中 +1
- Cl₂ **既是氧化剂又是还原剂**（歧化）

**例 6**：离子方程式

```
Cl₂ + 2OH⁻ → Cl⁻ + ClO⁻ + H₂O    （冷、稀 NaOH）
Cl₂ + 2I⁻ → I₂ + 2Cl⁻
Ag⁺ + Cl⁻ → AgCl↓
```

**例 7**：粗盐除杂中的离子反应

```
Ba²⁺ + SO₄²⁻ → BaSO₄↓
Ca²⁺ + CO₃²⁻ → CaCO₃↓
Mg²⁺ + 2OH⁻ → Mg(OH)₂↓
```

---

### 三、拓展

- **氧化性递变**：F₂ > Cl₂ > Br₂ > I₂（同主族上→下减弱）
- **还原性递变**：F⁻ < Cl⁻ < Br⁻ < I⁻（与单质氧化性相反）
- **离子反应 vs 复分解**：复分解是离子反应的一种（生成沉淀/气体/水）

---

### 四、易错点（2.2）

| 说法 | 正误 | 解析 |
|------|------|------|
| 有单质参加一定是氧化还原 | ❌ | 同素异形体转化（O₂→O₃）无电子转移 |
| HClO 在离子方程式中拆成 H⁺ + ClO⁻ | ❌ | 弱电解质，写 **HClO** |
| 写离子方程式忽略电荷守恒 | ❌ | 必须检查两边电荷相等 |
| Cl₂ 与 NaOH（热、浓）产物同冷稀 | ❌ | 热浓：NaCl + NaClO₃ + H₂O（不要求配平，了解即可） |

## 2.3 溴和碘的提取

### 一、概念

#### 1. 卤素单质性质

| 单质 | 常温状态 | 颜色 | 密度（相对） |
|------|----------|------|--------------|
| F₂ | 气体 | 淡黄 | 轻 |
| Cl₂ | 气体 | 黄绿 | > 空气 |
| Br₂ | 液体 | 深红棕 | > 水 |
| I₂ | 固体 | 紫黑 | — |

**氧化性递变：** F₂ > Cl₂ > Br₂ > I₂

**与 H₂ 反应剧烈程度：** F₂ > Cl₂ > Br₂ > I₂（越来越难）

#### 2. 提取方法

**氯气置换法（从卤水/海水）：**

```
Cl₂ + 2Br⁻ → Br₂ + 2Cl⁻
Cl₂ + 2I⁻ → I₂ + 2Cl⁻
```

**海带提碘：**

1. 海带浸泡、过滤得含 I⁻ 溶液
2. 通 Cl₂ 或加氧化剂氧化 I⁻ → I₂
3. **萃取**（CCl₄ 或苯）→ 分液得 I₂ 有机层

#### 3. 卤素离子检验

| 离子 | 试剂 | 沉淀颜色 |
|------|------|----------|
| Cl⁻ | AgNO₃ + 稀 HNO₃ | 白色 AgCl |
| Br⁻ | AgNO₃ + 稀 HNO₃ | 淡黄色 AgBr |
| I⁻ | AgNO₃ + 稀 HNO₃ | 黄色 AgI |

**AgCl、AgBr、AgI** 均不溶于稀硝酸，颜色逐渐加深。

---

### 二、举例

**例 8**：向 NaBr 溶液中通 Cl₂，溶液变橙色（Br₂ 溶于水呈橙色）。

**例 9**：向 KI 溶液中加 CCl₄ 和 Cl₂，振荡后 CCl₄ 层呈**紫红色**（I₂）。

**例 10**：鉴别 Cl⁻、Br⁻、I⁻ 三种溶液——分别加 AgNO₃ + HNO₃，看沉淀颜色。

---

### 三、拓展

- **溴的用途**：制药、照相、阻燃剂等；Br₂ 易挥发，需密封保存
- **碘的用途**：碘酒消毒、加碘盐（KIO₃ 或 KI）、甲状腺激素成分
- **萃取提碘**：利用 I₂ 在 CCl₄ 中溶解度远大于在水中

---

### 四、易错点（2.3）

| 说法 | 正误 | 解析 |
|------|------|------|
| Br₂ 是红棕色气体 | ❌ | 常温下是**液体** |
| I₂ 使淀粉变蓝是化学变化 | ✅ | I₂ 与淀粉形成包合物 |
| 用 Br₂ 从 I⁻ 溶液中提取 I₂ | ❌ | Br₂ 氧化性 < I₂ 不能氧化 I⁻；需 **Cl₂** |
| 卤素单质均易溶于水 | ❌ | Cl₂ 可溶；Br₂、I₂ 在水中溶解度较小 |

---

## 五、本章速记卡片

```
粗盐除杂顺序：BaCl₂ → Na₂CO₃ → NaOH → 过滤 → HCl → 蒸发

电解食盐水：2NaCl + 2H₂O → 2NaOH + Cl₂ + H₂

卤素氧化性：F₂ > Cl₂ > Br₂ > I₂
置换提取：Cl₂ + 2X⁻ → X₂ + 2Cl⁻（X = Br, I）

AgCl（白）→ AgBr（淡黄）→ AgI（黄）  均不溶于稀 HNO₃

Cl₂ 歧化：Cl₂ + 2NaOH → NaCl + NaClO + H₂O
漂白：HClO 强氧化性（氯水、漂白粉）
```
