# 氧化反应与还原反应

## 定义（初中简化）

| 类型 | 简化理解 |
|------|----------|
| **氧化反应** | 得氧 / 失电子 |
| **还原反应** | 失氧 / 得电子 |

## 典型例子

- **氧化**：C + O₂ → CO₂（C 得氧）
- **还原**：CO + Fe₂O₃ → Fe + CO₂（Fe₂O₃ 失氧，CO 得氧）

## 还原剂还原能力（炼铁）

C（高温）≈ CO（高温）> H₂

工业炼铁：3CO + Fe₂O₃ → 2Fe + 3CO₂（高温）

## 燃烧条件

可燃物 + 氧气 + 温度达到着火点（灭火：破坏任一条件）

## 高一深化（氧化还原反应）

| 概念 | 要点 |
|------|------|
| **氧化剂** | 得电子，化合价**降低**，被还原 |
| **还原剂** | 失电子，化合价**升高**，被氧化 |
| **判断** | 升失氧还，降得氧氧（化合价升高失电子被氧化，降低得电子被还原） |

**常见氧化剂：** Cl₂、O₂、KMnO₄、浓 H₂SO₄、HNO₃

**歧化反应：** 同一元素既升价又降价，如 Cl₂ + 2NaOH → NaCl + NaClO + H₂O
