# 初高中化学知识库

欢迎使用 Karpathy LLM Wiki 模式的化学知识库。

## 多端分工

| 设备 | 角色 |
|------|------|
| Mac / Windows | 收集、ingest、git push |
| 手机 / 平板 | 只读、看 Graph |

详细说明 → [[多端同步说明]]

## 如何使用

1. **Obsidian**：将此文件夹作为 Vault 打开，用于阅读和浏览图谱
2. **Cursor / Claude Code**：在此目录打开项目，与 Agent 对话执行 ingest / query / lint
3. **规则文件**：Agent 会自动读取根目录的 `CLAUDE.md`

## 三条命令

| 命令 | 说明 | 示例 |
|------|------|------|
| **ingest** | 收录新资料到 raw/ 并编译 wiki | 「ingest raw/inbox/ 里的文件」 |
| **query** | 基于 wiki 提问 | 「氧化还原反应怎么判断？」 |
| **lint wiki** | 健康检查断链、矛盾、缺口 | 「lint wiki」 |

## 目录说明

- `raw/ella/` — Ella 化学资料镜像
- `raw/inbox/` — 待 ingest 的新资料
- `raw/assets/` — 图片、PDF 附件
- `wiki/` — 阅读与 Graph 主战场（summaries、concepts、entities…）
