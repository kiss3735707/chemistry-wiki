# 初中化学溶解性表 · 摘要

判断酸、碱、盐在水中溶解能力及沉淀颜色的核心工具，**复分解反应必备**。

## 溶解性四等级（20℃，100 g 水）

| 等级 | 范围 |
|------|------|
| 易溶 | > 10 g |
| 可溶 | 1~10 g |
| 微溶 | 0.01~1 g（复分解中常作沉淀） |
| 难溶 | < 0.01 g |

## 必背口诀

```
钾钠铵硝皆易溶；氯除银铅；硫除钡铅钙；
碳磷碳酸多不溶；碱除钾钠钡钙微
```

## 关键例外

- AgCl、BaSO₄：**不溶于稀酸**（Cl⁻、SO₄²⁻ 专属检验）
- Ca(OH)₂：**微溶**但可作碱用
- Na₂CO₃：是**盐**不是碱，溶液呈碱性
