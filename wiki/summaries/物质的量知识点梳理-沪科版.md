# 物质的量知识点梳理 · 摘要（沪科版）

沪科版必修第一册 **1.2 物质的量** 专题梳理，含概念、计算、实验衔接与易错点。

## 教材定位

- 第 1 章 化学研究的天地 · 1.2 物质的量
- 关联 1.3：配制一定物质的量浓度的溶液、测定气体摩尔体积

## 核心公式

```
n = m/M = N/N_A = V/22.4（标况气体）
c = n/V；稀释 c₁V₁ = c₂V₂
```

## 五大物理量

n、N_A、M、V_m（标况 22.4 L/mol）、c（mol/L）

## 常考题型

1. 粒子数 ↔ n（注意化学式中原子倍数）
2. 质量 ↔ n
3. 标况气体 V ↔ n
4. 浓度配制与稀释
5. 混合气体（同温同压 V₁/V₂ = n₁/n₂）
6. 方程式综合 + 密度求 M（M = 22.4ρ）

## 易错点

- 22.4 仅用于标况**气体**
- c = n/V 中 V 是溶液体积
- 64 g O₂ = 4 mol O 原子（非 2 mol）
