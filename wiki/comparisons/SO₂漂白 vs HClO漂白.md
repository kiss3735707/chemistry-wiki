# SO₂漂白 vs HClO漂白

## 对比

| | SO₂ 漂白 | HClO 漂白 |
|---|---------|-----------|
| **代表物质** | [[entities/二氧化硫]] | HClO（[[entities/氯气]] 与水反应生成） |
| **原理** | 与有色物质**化合** | **强氧化性**破坏有色物质 |
| **是否可逆** | 加热可恢复原色 | 不可逆 |
| **能否漂白石蕊** | 先变红后褪色（溶于水呈酸性） | 先变红后褪色（氧化） |
| **持久性** | 不持久（加热恢复） | 持久 |

## 要点

- 两者都能漂白某些有色物质，但**原理完全不同**
- 考试常考：SO₂ 使品红褪色，加热恢复 → 化合型漂白
- HClO / 氯水漂白 → 氧化型，不可逆
