# SO₂漂白 vs HClO漂白

## 对比

| | SO₂ 漂白 | HClO 漂白 |
|---|---------|-----------|
| **代表物质** | [[entities/二氧化硫]] | HClO（[[entities/氯气]] 与水反应生成） |
| **反应类型** | **化合反应**（加成） | **氧化还原反应** |
| **原理** | 与有色物质**结合**生成无色不稳定化合物 | **强氧化性**氧化破坏发色基团 |
| **是否可逆** | 加热可恢复原色 | 不可逆 |
| **典型检验/现象** | 品红褪色，**加热恢复红色** | 湿润有色布条褪色，**加热不恢复** |
| **持久性** | 不持久 | 持久 |

---

## 化学反应方式

### SO₂ 漂白 — 化合型（非氧化还原）

**本质：** SO₂ 与某些有色有机物（如品红）发生**加成/化合**，生成**无色且不稳定的化合物**；并非靠强氧化性破坏分子结构。

**涉及反应：**

```
SO₂ + H₂O ⇌ H₂SO₃              （SO₂ 先溶于水，提供漂白活性）
SO₂ + 有色物质 ⇌ 无色化合产物     （如品红被漂白）
```

**加热时逆反应：**

```
无色化合产物 → SO₂↑ + 有色物质    （加热分解，红色恢复）
```

**要点：**
- 漂白对象是**特定有色物质**（品红等），不能漂白所有有色物质
- **不能**使紫色石蕊褪色（SO₂ 使石蕊变红是**酸性**，不是漂白）
- SO₂ 虽有还原性，但品红漂白**不属于**氧化还原漂白

---

### HClO 漂白 — 氧化型

**本质：** HClO 具有**强氧化性**，将有色物质**氧化**为无色或浅色产物，发色基团被破坏，**不可逆**。

**HClO 的来源：**

```
Cl₂ + H₂O ⇌ HCl + HClO              （氯水、湿润 Cl₂ 漂白）
Ca(ClO)₂ + CO₂ + H₂O → CaCO₃ + 2HClO   （漂白粉遇 CO₂ 释放 HClO）
NaClO + H₂O ⇌ NaOH + HClO           （84 消毒液）
```

**漂白反应（示意）：**

```
HClO + 有色物质 → 氧化产物（无色/浅色） + HClO 被还原
```

**具体表现：**
- **湿润**有色布条：Cl₂ 先与水生成 HClO，再氧化漂白
- **干燥**有色布条：**不褪色**（无 HClO 生成）
- 石蕊：先变红（H⁺），后褪色（HClO 氧化）

**HClO 自身分解（与漂白无关，但需了解）：**

```
2HClO → 2HCl + O₂↑    （见光分解，需避光保存）
```

## 鉴别方法

| 操作 | SO₂ | HClO / 氯水 |
|------|-----|-------------|
| 通入品红 | 褪色 | 褪色 |
| **加热** | **红色恢复** | 仍无色 |
| 通入酸性 KMnO₄ | 溶液褪色（SO₂ 被氧化） | 一般不明显 |

**记忆：** 品红 + 加热 → 能恢复的是 **SO₂**。

---

## 易错点

| 说法 | 正误 | 解析 |
|------|------|------|
| SO₂ 漂白是氧化还原 | ❌ | 品红漂白是**化合/加成**，可逆 |
| SO₂ 使石蕊褪色 | ❌ | 石蕊变红是**酸性**；SO₂ 不能漂白石蕊 |
| 干燥 Cl₂ 能漂有色布条 | ❌ | 需**湿润**（Cl₂ + H₂O → HClO） |
| 两种漂白原理相同 | ❌ | 化合 vs 氧化，**本质不同** |
