# 氨

## 基本信息

- **化学式**：NH₃
- **类别**：氢化物 / 气体
- **颜色**：无色，有刺激性气味

## 性质与用途

- **物理**：极易溶于水（1:700）；密度比空气小
- **化学**：碱性气体；与 HCl 反应生成白烟（NH₄Cl）
- **用途**：制硝酸、铵盐、化肥；工业合成氨

## 相关反应

```
N₂ + 3H₂ ⇌ 2NH₃              （高温高压，催化剂，工业合成氨）
NH₃ + HCl → NH₄Cl            （白烟）
NH₃ + H₂O ⇌ NH₃·H₂O          （弱碱）
NH₄⁺ + OH⁻ → NH₃↑ + H₂O     （铵盐检验）
```
