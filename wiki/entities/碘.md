# 碘

## 基本信息

- **化学式**：I₂
- **类别**：卤素单质
- **常温状态**：紫黑色**固体**，易升华

## 性质与用途

- **物理**：密度较大；难溶于水，易溶于有机溶剂；易升华
- **化学**：氧化性 Br₂ > **I₂**；与淀粉溶液显**蓝色**（特征反应）
- **用途**：碘酒消毒、加碘盐、甲状腺激素成分；海带提碘

## 相关反应

```
Cl₂ + 2I⁻ → I₂ + 2Cl⁻              （氯气置换法 / 海带提碘）
I₂ + 淀粉 → 蓝色                    （检验 I₂）
Br₂ + 2I⁻ → I₂ + 2Br⁻              （Br₂ 可氧化 I⁻）
```

**检验 I⁻：** AgNO₃ + HNO₃ → 黄色 AgI 沉淀

**海带提碘流程：** 浸泡 → 过滤 → 通 Cl₂ 氧化 → **萃取 + 分液**（CCl₄ 层呈紫红色）
