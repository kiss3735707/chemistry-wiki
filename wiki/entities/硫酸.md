# 硫酸

## 基本信息

- **化学式**：H₂SO₄
- **类别**：强酸（含氧酸）
- **特性**：浓硫酸有吸水性、脱水性、强氧化性

## 性质与用途

### 稀硫酸

- 体现 H⁺ 的通性：与活泼金属（H 前）、碱、碱性氧化物、某些盐反应
- **不与 Cu、Ag 等不活泼金属反应**

### 浓硫酸三大特性

| 特性 | 实例 |
|------|------|
| **吸水性** | 干燥 H₂、CO₂；使 CuSO₄·5H₂O 变白 |
| **脱水性** | 使蔗糖、棉花炭化（按 2:1 脱 H、O） |
| **强氧化性**（加热） | 与 Cu、C 反应生成 SO₂；常温**钝化** Fe、Al |

## 相关反应

```
Cu + 2H₂SO₄(浓) → CuSO₄ + SO₂↑ + 2H₂O    （加热）
C + 2H₂SO₄(浓) → CO₂↑ + 2SO₂↑ + 2H₂O     （加热）
2SO₂ + O₂ ⇌ 2SO₃ → H₂SO₄                  （工业制硫酸）
```

**稀释浓硫酸：** **酸入水**，沿器壁缓慢加入，不断搅拌。
