# 溴

## 基本信息

- **化学式**：Br₂
- **类别**：卤素单质
- **常温状态**：深红棕色**液体**，易挥发

## 性质与用途

- **物理**：密度比水大；微溶于水，易溶于有机溶剂（CCl₄、苯）；有毒、有刺激性
- **化学**：氧化性 Cl₂ > **Br₂** > I₂；可与 Fe、H₂ 等反应
- **用途**：制药、照相、阻燃剂；从海水/卤水中提取

## 相关反应

```
Cl₂ + 2Br⁻ → Br₂ + 2Cl⁻          （氯气置换法提溴）
2Fe + 3Br₂ → 2FeBr₃
Br₂ + H₂ → 2HBr                   （加热或光照）
```

**检验 Br⁻：** AgNO₃ + HNO₃ → 淡黄色 AgBr 沉淀
