# 二氧化硫

## 基本信息

- **化学式**：SO₂
- **类别**：酸性氧化物 / 气体
- **颜色**：无色，有刺激性气味

## 性质与用途

- **物理**：易溶于水（可逆）；密度比空气大
- **化学**：酸性氧化物；**漂白性**（与有色物质化合生成无色不稳定物，加热可恢复）；还原性
- **环境**：形成酸雨的前体物质

## 相关反应

```
SO₂ + H₂O ⇌ H₂SO₃
SO₂ + O₂ → SO₃（催化剂，加热）
2SO₂ + O₂ ⇌ 2SO₃（工业制硫酸）
```
