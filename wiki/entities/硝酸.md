# 硝酸

## 基本信息

- **化学式**：HNO₃
- **类别**：强酸（含氧酸）
- **特性**：强氧化性

## 性质与用途

- **物理**：无色液体；浓硝酸见光分解（保存在棕色瓶中）
- **化学**：**强氧化性**；浓/稀硝酸与 Cu 反应；常温下**钝化** Al、Fe
- **用途**：制炸药、染料、化肥

## 相关反应

```
4HNO₃ → 4NO₂↑ + O₂↑ + 2H₂O   （浓硝酸见光分解）
3NO₂ + H₂O → 2HNO₃ + NO       （NO₂ 与水）
3Cu + 8HNO₃(稀) → 3Cu(NO₃)₂ + 2NO↑ + 4H₂O
Cu + 4HNO₃(浓) → Cu(NO₃)₂ + 2NO₂↑ + 2H₂O
```
