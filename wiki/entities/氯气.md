# 氯气

## 基本信息

- **化学式**：Cl₂
- **类别**：卤素单质 / 气体
- **颜色**：黄绿色，有毒，有刺激性气味

## 性质与用途

- **物理**：密度比空气大，可溶于水
- **化学**：**强氧化性**；与 Fe、Cu、H₂、NaOH、H₂O 等反应
- **用途**：消毒、漂白、制盐酸、提取 Br/I

## 相关反应

```
Cl₂ + 2NaOH → NaCl + NaClO + H₂O    （冷、稀，制漂白液）
Cl₂ + H₂O ⇌ HCl + HClO               （歧化，漂白原理）
2NaCl + 2H₂O → 2NaOH + Cl₂↑ + H₂↑   （电解饱和食盐水）
```

**次氯酸 HClO：** 弱酸、不稳定；漂白原理为**氧化有色物质**（不可逆）。详见 [[comparisons/SO₂漂白 vs HClO漂白]]。
