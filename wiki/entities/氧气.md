# 氧气

## 基本信息

- **化学式**：O₂
- **类别**：非金属单质 / 气体
- **空气中含量**：约 21%（体积分数）

## 性质

- **物理**：无色无味，密度略大于空气
- **化学**：助燃性（氧化性）；与 C、S、P、Fe 等反应

## 制取

| 方法 | 方程式 |
|------|--------|
| 加热 KMnO₄ | 2KMnO₄ → K₂MnO₄ + MnO₂ + O₂↑ |
| 分解 H₂O₂ | 2H₂O₂ → 2H₂O + O₂↑（MnO₂ 催化） |
| 工业 | 分离液态空气（**物理变化**） |

## 验满

带火星木条放在集气瓶**口**，复燃则满。
