# 硫

## 基本信息

- **化学式**：S
- **类别**：非金属单质
- **颜色**：淡黄色固体

## 性质与用途

- **物理**：难溶于水，微溶于酒精，易溶于 CS₂
- **化学**：与 O₂ 生成 SO₂；与 Fe、Cu 生成硫化物；与 H₂ 生成 H₂S
- **用途**：制 H₂SO₄、橡胶硫化、农药、火药

## 相关反应

```
S + O₂ → SO₂              （点燃）
Fe + S → FeS              （加热）
Cu + S → CuS              （加热）
H₂ + S → H₂S              （加热）
```
