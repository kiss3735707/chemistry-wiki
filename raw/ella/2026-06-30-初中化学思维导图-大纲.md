# 初中化学知识框架

## 一、化学基本概念

### 走进化学世界
- 化学研究对象：物质及其变化
- 物理变化 vs 化学变化（是否有新物质）
- 物理性质 vs 化学性质
- 实验基本操作（取用、加热、过滤、蒸发）
- 探究步骤：问题→假设→实验→结论

### 物质分类
- 纯净物
  - 单质（金属、非金属）
  - 化合物
    - 氧化物
    - 酸
    - 碱
    - 盐
- 混合物（空气、溶液、合金）

### 化学用语
- 元素符号
- 化学式（读写、含义、化合价）
- 化学方程式（书写、配平、意义）
- 离子符号（H⁺、OH⁻、Na⁺、Cl⁻、CO₃²⁻）

### 微观粒子
- 分子：保持化学性质的最小粒子
- 原子：化学变化中最小粒子
- 离子：带电的原子或原子团
- 化学变化中分子可分、原子不可分

## 二、空气和氧气

### 空气
- 成分：N₂(78%)、O₂(21%)、稀有气体、CO₂
- 测定O₂含量：红磷燃烧法
- 空气污染：SO₂、NO₂、颗粒物
- 防治：清洁能源、尾气处理

### 氧气
- 物理性质：无色无味，密度略大于空气
- 化学性质：助燃性（氧化性）
- 实验室制取
  - 加热KMnO₄
  - 分解H₂O₂（MnO₂催化）
  - 加热KClO₃（MnO₂催化）
- 工业制取：分离液态空气
- 验满：带火星木条放瓶口复燃
- 用途：医疗、炼钢、气焊

## 三、水和氢

- 水的组成：电解水 H₂:O₂ = 2:1
- 水的净化：沉淀→过滤→吸附→蒸馏
- 硬水与软水：肥皂水区分，煮沸/蒸馏软化
- 氢气
  - 最轻气体
  - 可燃性（点燃前验纯）
  - 还原性
  - 制取：Zn + H₂SO₄ → ZnSO₄ + H₂↑

## 四、碳和碳的氧化物

### 碳单质（同素异形体）
- 金刚石：硬度大
- 石墨：导电、润滑
- C₆₀：新材料

### 碳的化学性质
- 可燃性（充分→CO₂，不充分→CO）
- 还原性（与CuO、Fe₂O₃反应）

### 二氧化碳 CO₂
- 不燃烧不支持燃烧
- 与水、碱反应
- 制取：CaCO₃ + 2HCl
- 检验：澄清石灰水变浑浊
- 验满：燃着木条放瓶口熄灭
- 用途：灭火、干冰、光合作用
- 温室气体

### 一氧化碳 CO
- 有毒
- 可燃性、还原性
- 炼铁：CO + Fe₂O₃ → Fe + CO₂

## 五、金属

- 物理性质：导电导热、光泽、延展性
- 化学性质
  - 与O₂反应
  - 与稀酸反应（置换）
  - 与盐溶液反应（置换）
- 金属活动性顺序
  - K Ca Na Mg Al Zn Fe Sn Pb (H) Cu Hg Ag Pt Au
  - 前置换后
  - H前金属与稀酸反应
- 铁的冶炼：3CO + Fe₂O₃ → 2Fe + 3CO₂
- 金属保护：防锈、回收、合理开采

## 六、溶液

- 组成：溶质 + 溶剂
- 溶解（物理过程）与乳化
- 饱和溶液
- 溶解度（100g水中，受温度影响）
- 溶质质量分数 = 溶质/溶液 × 100%
- 配制溶液：计算→称量→溶解→装瓶
- 溶解度曲线：查溶解度、判断饱和、结晶

## 七、酸和碱

### 酸（H⁺）
- 常见：HCl、H₂SO₄、HNO₃、H₂CO₃
- 通性：指示剂、金属、碱、盐
- 浓H₂SO₄：吸水性、脱水性、酸入水稀释

### 碱（OH⁻）
- 常见：NaOH、Ca(OH)₂
- 通性：指示剂、酸、盐
- NaOH潮解、Ca(OH)₂微溶

### 中和反应
- 酸 + 碱 → 盐 + 水（放热）
- 应用：改良土壤、处理废水、胃药

## 八、盐

- NaCl（食盐）
- Na₂CO₃（纯碱）
- NaHCO₃（小苏打）
- CaCO₃（石灰石）
- CuSO₄（波尔多液）
- 复分解反应
  - AB + CD → AD + CB
  - 条件：生成水、气体或沉淀

## 九、化学与生活

- 有机物（含碳化合物）
- 三大营养素：糖类、油脂、蛋白质
- 维生素与微量元素
- 材料：塑料、合成纤维、合成橡胶
- 能源：化石燃料、新能源
- 环境保护与绿色化学

## 十、化学反应类型

- 化合反应：A + B → C
- 分解反应：A → B + C
- 置换反应：A + BC → AC + B
- 复分解反应：AB + CD → AD + CB
- 氧化反应与还原反应
- 燃烧条件：可燃物+O₂+着火点

## 十一、化学计算

- 相对分子质量
- 元素质量比与质量分数
- 化学方程式计算（质量守恒）
- 溶液计算（溶质质量分数、稀释）
- 图表题（溶解度曲线、反应曲线）

## 十二、实验探究

- O₂制取与性质
- 水的电解
- CO₂制取与性质
- 金属活动性探究
- 酸碱性质与中和

## 十三、鉴别与颜色

- 黑色固体：CuO、Fe₃O₄、C、MnO₂
- 蓝色溶液：CuSO₄
- 白色沉淀：BaSO₄、AgCl、CaCO₃
- 气体鉴别：O₂、CO₂、H₂、CO
