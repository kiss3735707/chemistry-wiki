## 一、溶解性四等级标准

**条件：** 20 ℃，100 g 水中

| 等级 | 溶解度范围 | 符号 | 说明 |
|------|------------|------|------|
| **易溶** | > 10 g | 易 | 大量溶解 |
| **可溶** | 1 ~ 10 g | 可 | 明显溶解 |
| **微溶** | 0.01 ~ 1 g | 微 | 少量溶解，常作沉淀处理 |
| **难溶 / 不溶** | < 0.01 g | 难 / 不 | 基本不溶，视为沉淀 |

**初中常用简化：** 除标注「微溶」外，「难溶」与「不溶」在解题中同等对待（均视为有沉淀）。

## 二、速记口诀（复分解反应必备)

```
钾钠铵硝皆易溶 —— K⁺、Na⁺、NH₄⁺、NO₃⁻ 的盐一般易溶
氯除银铅       —— Cl⁻ 盐：AgCl 不溶，PbCl₂ 微溶
硫除钡铅钙     —— SO₄²⁻ 盐：BaSO₄、PbSO₄ 不溶，CaSO₄ 微溶
碳磷碳酸多不溶 —— CO₃²⁻、PO₄³⁻ 盐：除 K⁺、Na⁺、NH₄⁺ 外多不溶
碱除钾钠钡钙微 —— 可溶碱：KOH、NaOH、Ba(OH)₂；Ca(OH)₂ 微溶
```

**酸：** 初中常见酸（HCl、H₂SO₄、HNO₃、H₂CO₃、CH₃COOH）均易溶于水。

## 三、常见酸溶解性表

| 化学式 | 名称 | 溶解性 | 溶液 / 状态颜色 |
|--------|------|--------|-----------------|
| HCl | 盐酸 | 易溶 | <span style="color:#666">无色</span>，有刺激性气味 |
| H₂SO₄ | 硫酸 | 易溶 | <span style="color:#666">无色</span>油状液体 |
| HNO₃ | 硝酸 | 易溶 | 稀硝酸 <span style="color:#666">无色</span>；浓硝酸 <span style="color:#DAA520">黄色</span>（久置分解产生 NO₂） |
| H₂CO₃ | 碳酸 | 易溶（不稳定） | <span style="color:#666">无色</span>，常温迅速分解为 CO₂ 和 H₂O |
| CH₃COOH | 醋酸 | 易溶 | <span style="color:#666">无色</span> |
| H₃PO₄ | 磷酸 | 易溶 | <span style="color:#666">无色</span> |
| H₂SiO₃ | 硅酸 | 难溶 | 白色胶状沉淀 |

## 四、常见碱溶解性表

| 化学式 | 名称 | 溶解性 | 溶液 / 沉淀颜色 |
|--------|------|--------|-----------------|
| NaOH | 氢氧化钠 | 易溶 | 溶液 <span style="color:#666">无色</span>；固体白色 |
| KOH | 氢氧化钾 | 易溶 | 溶液 <span style="color:#666">无色</span> |
| Ba(OH)₂ | 氢氧化钡 | 易溶 | 溶液 <span style="color:#666">无色</span> |
| Ca(OH)₂ | 氢氧化钙 | 微溶 | 溶液（石灰水）<span style="color:#666">无色</span>；固体白色 |
| NH₃·H₂O | 氨水 | 易溶 | 溶液 <span style="color:#666">无色</span>，有刺激性气味 |
| Cu(OH)₂ | 氢氧化铜 | 难溶 | <span style="color:#1E90FF">**不溶（蓝色沉淀）**</span> |
| Fe(OH)₃ | 氢氧化铁 | 难溶 | <span style="color:#8B4513">**不溶（红褐色沉淀）**</span> |
| Fe(OH)₂ | 氢氧化亚铁 | 难溶 | <span style="color:#666">**不溶（白色沉淀）**</span>，迅速变灰绿 → 红褐 |
| Mg(OH)₂ | 氢氧化镁 | 难溶 | <span style="color:#666">**不溶（白色沉淀）**</span> |
| Al(OH)₃ | 氢氧化铝 | 难溶 | <span style="color:#666">**不溶（白色沉淀）**</span>，两性，溶于强酸和强碱 |

**碱颜色口诀：** 可溶碱溶液皆无色；难溶碱看沉淀 — <span style="color:#1E90FF">Cu(OH)₂ 蓝</span>、<span style="color:#8B4513">Fe(OH)₃ 红褐</span>、其余多为白。

## 五、常见盐溶解性规律表

### 5.1 阳离子规律

| 阳离子 | 溶解性规律 | 典型例外 / 备注 |
|--------|------------|-----------------|
| **K⁺、Na⁺、NH₄⁺** | 绝大多数盐 **易溶** | 无常见难溶例外 |
| **Ag⁺** | 多数 **难溶** | AgNO₃ 易溶；AgCl、Ag₂SO₄（微）、Ag₂CO₃ 等难溶 |
| **Ba²⁺** | 多数 **难溶** | BaCl₂、Ba(NO₃)₂ 易溶；BaSO₄、BaCO₃ 难溶 |
| **Ca²⁺** | 多数 **微溶或难溶** | CaCl₂、Ca(NO₃)₂ 易溶；CaSO₄ 微溶；CaCO₃ 难溶 |
| **Mg²⁺、Al³⁺、Fe²⁺、Fe³⁺、Cu²⁺** | 氯化物、硝酸盐多易溶 | 碳酸盐、磷酸盐、氢氧化物多难溶 |

### 5.2 阴离子规律

| 阴离子 | 溶解性规律 | 典型例外 |
|--------|------------|----------|
| **NO₃⁻** | 所有硝酸盐 **易溶** | 无例外 |
| **Cl⁻** | 大多数 **易溶** | AgCl <span style="color:#666">不溶（白）</span>；PbCl₂ 微溶 |
| **SO₄²⁻** | 大多数 **易溶** | BaSO₄ <span style="color:#666">不溶（白）</span>；CaSO₄ 微溶；PbSO₄ 不溶 |
| **CO₃²⁻** | 大多数 **难溶** | K₂CO₃、Na₂CO₃、(NH₄)₂CO₃ 易溶 |
| **HCO₃⁻** | 大多数 **易溶** | NaHCO₃、Ca(HCO₃)₂ 等 |
| **PO₄³⁻** | 大多数 **难溶** | K₃PO₄、Na₃PO₄、(NH₄)₃PO₄ 易溶 |
| **OH⁻** | 见 [碱溶解性表](#四常见碱溶解性表) | — |

| 符号 | 含义 |

| **溶** | 易溶 / 可溶 |

### 6.3 网格表速查要点

```
① K⁺、Na⁺、NH₄⁺ 与各阴离子交叉 → 几乎全「溶」
② NO₃⁻ 列 → 全「溶」（无例外）
③ Cl⁻ 列 → Ag⁺「不」、Pb²⁺「微」，其余多「溶」
④ SO₄²⁻ 列 → Ba²⁺、Pb²⁺「不」，Ca²⁺「微」，其余多「溶」
⑤ CO₃²⁻ / PO₄³⁻ 列 → 仅 K⁺、Na⁺、NH₄⁺「溶」，其余多「不」
⑥ OH⁻ 行 → 仅 K⁺、Na⁺、Ba²⁺、NH₄⁺「溶」；Ca²⁺「微」；Cu²⁺「不」蓝；Fe³⁺「不」红褐
```

## 八、离子检验与沉淀颜色对照

| 检验离子 | 常用试剂 | 溶解性 | 现象（颜色） |
|----------|----------|--------|--------------|
| Cl⁻ | AgNO₃ + HNO₃ | AgCl 难溶 | <span style="color:#666">白色沉淀</span> |
| SO₄²⁻ | BaCl₂ + HCl | BaSO₄ 难溶 | <span style="color:#666">白色沉淀</span> |
| CO₃²⁻ | 稀盐酸 | 生成 CO₂ | 气泡，澄清石灰水变浑浊 |
| NH₄⁺ | NaOH 加热 | NH₃ 易溶 | 刺激性气体，湿润石蕊试纸变蓝 |
| Fe³⁺ | NaOH | Fe(OH)₃ 难溶 | <span style="color:#8B4513">红褐色沉淀</span> |
| Fe²⁺ | NaOH | Fe(OH)₂ 难溶 | <span style="color:#666">白色沉淀</span> → 灰绿 → 红褐 |
| Cu²⁺ | NaOH | Cu(OH)₂ 难溶 | <span style="color:#1E90FF">蓝色沉淀</span> |
| Ca²⁺ | Na₂CO₃ | CaCO₃ 难溶 | <span style="color:#666">白色沉淀</span> |
| Ba²⁺ | H₂SO₄ / Na₂SO₄ | BaSO₄ 难溶 | <span style="color:#666">白色沉淀</span> |
| Ag⁺ | HCl | AgCl 难溶 | <span style="color:#666">白色沉淀</span> |

## 九、易错提醒

| 易错点 | 正确理解 |
|--------|----------|
| Ca(OH)₂ 是「可溶碱」吗？ | **微溶**，但石灰水可用来检验 CO₂，作碱用 |
| Na₂CO₃ 是碱吗？ | **不是**，是盐；水溶液呈碱性 |
| CuSO₄ 固体 vs 溶液 | 无水 CuSO₄ 为 <span style="color:#666">白色</span>；CuSO₄·5H₂O 胆矾为 <span style="color:#1E90FF">蓝色</span>；溶于水为 <span style="color:#1E90FF">蓝色溶液</span> |
| 微溶算不算沉淀？ | 复分解反应中 **CaSO₄、Ag₂SO₄、PbCl₂、Ca(OH)₂** 等微溶物 **视为有沉淀** |
| AgCl、BaSO₄ 加酸 | **不溶于稀酸**，可用于 Cl⁻、SO₄²⁻ 的专属检验 |
| CaCO₃、BaCO₃ 加酸 | **溶于酸**并产生 CO₂ 气泡，可区别于 AgCl、BaSO₄ |
