## 三、分章知识框架

### 第 1 章 · 化学研究的天地

#### 1.1 物质的分类

| 知识点 | 要点 |
|--------|------|
| 分类方法 | 交叉分类法、树状分类法 |
| 物质分类树 | 纯净物 / 混合物；单质 / 化合物；酸 / 碱 / 盐 / 氧化物 |
| 分散系 | 溶液、胶体、浊液；**丁达尔效应** |
| 胶体 | 氢氧化铁胶体制备；胶体与溶液、浊液的区别 |

**易错点：** 胶体是分散系，不是纯净物；按分散质粒子直径（1～100 nm）判断胶体。

#### 1.2 物质的量

| 知识点 | 要点 |
|--------|------|
| 基本概念 | 物质的量 n、阿伏加德罗常数 N_A、摩尔 mol |
| 摩尔质量 | M = m/n；相对分子/原子质量数值上等于 M（g/mol） |
| 气体摩尔体积 | 标准状况下 V_m ≈ 22.4 L/mol（**仅适用于气体**） |
| 物质的量浓度 | c = n/V（V 为溶液体积，单位 L） |
| 换算关系 | n = m/M = N/N_A = V/V_m（气体，标况） |

**核心公式链：**

```
质量 m ──÷M──→ 物质的量 n ──×N_A──→ 粒子数 N
                    │
                    ├──×V_m──→ 气体体积 V（标况）
                    └──÷V──→ 物质的量浓度 c
```

**易错点：**
- 22.4 L/mol 不适用于液体、固体
- 溶液稀释：c₁V₁ = c₂V₂

**活动：** 测定气体摩尔体积

#### 1.3 化学中常用的实验方法

| 类别 | 内容 |
|------|------|
| 物质的制备 | 气体发生装置选择（固+固加热、固+液不加热、固+液加热等） |
| 分离与提纯 | 过滤、蒸发、蒸馏、分液、萃取、重结晶 |
| 物质检验 | 常见阳离子（Na⁺、K⁺、NH₄⁺、Ba²⁺ 等）、阴离子（Cl⁻、SO₄²⁻、CO₃²⁻ 等） |
| 溶液配制 | **配制一定物质的量浓度的溶液**（计算、称量/量取、定容、误差分析） |

**高频考点：** 实验操作顺序、仪器选择、误差分析（仰视/俯视读数对 c 的影响）

### 第 2 章 · 海洋中的卤素资源

#### 2.1 海水中的氯

| 知识点 | 要点 |
|--------|------|
| 粗盐提纯 | 溶解 → 过滤 → 除杂（BaCl₂、Na₂CO₃、NaOH 顺序）→ 过滤 → 盐酸 → 蒸发 |
| 电解饱和食盐水 | 2NaCl + 2H₂O → 2NaOH + Cl₂↑ + H₂↑ |
| 氯气性质 | 黄绿色、有毒、有刺激性；**强氧化性** |
| 氯气反应 | 与 Fe、Cu、H₂、NaOH、水 等 |
| 次氯酸 / 次氯酸盐 | HClO 弱酸、不稳定；漂白粉成分与漂白原理 |

**重要方程式：**

```
Cl₂ + 2NaOH → NaCl + NaClO + H₂O    （冷、稀）
Cl₂ + H₂O ⇌ HCl + HClO               （可逆，歧化）
Ca(ClO)₂ + CO₂ + H₂O → CaCO₃ + 2HClO （漂白原理）
```

#### 2.2 氧化还原反应和离子反应

| 模块 | 要点 |
|------|------|
| **氧化还原反应** | 有电子转移；化合价升降；氧化剂得电子、还原剂失电子 |
| 氧化剂 / 还原剂 | 常见氧化剂：Cl₂、O₂、KMnO₄、浓 H₂SO₄、HNO₃ 等 |
| 氧化数 | 判断氧化还原；配平氧化还原方程式 |
| **电解质电离** | 强电解质 / 弱电解质；强酸强碱盐完全电离 |
| **离子反应** | 有离子参加的反应；发生条件：沉淀、气体、弱电解质、氧化还原 |
| 离子方程式 | 写、改、删、查四步；注意原子守恒与电荷守恒 |

**易错点：**
- 微溶物（如 CaSO₄）在离子方程式中的处理
- 次氯酸、氨水等弱电解质写分子式
- 歧化反应（如 Cl₂ + NaOH）

#### 2.3 溴和碘的提取

| 知识点 | 要点 |
|--------|------|
| 卤素单质 | F₂、Cl₂、Br₂、I₂ 颜色与状态；氧化性 F₂ > Cl₂ > Br₂ > I₂ |
| 提取方法 | 氯气置换法（Br⁻、I⁻）；海带提碘 |
| 卤素离子检验 | Cl⁻（AgNO₃ + HNO₃）；Br⁻、I⁻（AgNO₃ + HNO₃，沉淀颜色不同） |

### 第 3 章 · 硫、氮及其循环

#### 3.1 硫及其重要化合物

| 物质 | 要点 |
|------|------|
| 硫单质 | 黄色固体；与 O₂、Fe、H₂ 等反应 |
| 二氧化硫 SO₂ | 无色有刺激性；可逆溶于水；**漂白性**（与 HClO 漂白原理对比） |
| 三氧化硫 SO₃ | 易溶于水生成 H₂SO₄ |
| 硫酸 H₂SO₄ | 浓硫酸特性：**吸水性、脱水性、强氧化性** |
| 浓硫酸性质实验 | 与 Cu、C 反应（需加热）；钝化 Fe、Al |

**SO₂ 漂白 vs HClO 漂白：**

| | SO₂ 漂白 | HClO 漂白 |
|---|---------|-----------|
| 原理 | 与有色物质化合 | 强氧化性 |
| 是否可逆 | 加热可恢复 | 不可逆 |

#### 3.2 氮及其重要化合物

| 物质 | 要点 |
|------|------|
| 氮气 N₂ | 稳定；工业合成氨 N₂ + 3H₂ ⇌ 2NH₃ |
| 氨 NH₃ | 易溶于水；碱性；喷泉实验 |
| 铵盐 | 与碱共热放 NH₃；检验 NH₄⁺ |
| 一氧化氮 NO | 无色 → 遇 O₂ 变红棕色 NO₂ |
| 二氧化氮 NO₂ | 红棕色；3NO₂ + H₂O → 2HNO₃ + NO |
| 硝酸 HNO₃ | 强氧化性；浓/稀与 Cu 反应；钝化 Al、Fe |

#### 3.3 硫循环与氮循环

| 主题 | 要点 |
|------|------|
| 酸雨 | SO₂、NO_x 排放 → 硫酸型 / 硝酸型酸雨 |
| 防治 | 脱硫、尾气处理、清洁能源 |

**活动：** 测定硫酸铜晶体中结晶水的含量

### 第 4 章 · 原子结构和化学键

#### 4.1 元素周期表和元素周期律

| 知识点 | 要点 |
|--------|------|
| 周期表结构 | 7 周期、18 族；短周期（1–3 周期） |
| 原子序数关系 | 原子序数 = 质子数 = 核电荷数 |
| 元素周期律 | 同周期：左→右，金属性↓、非金属性↑、原子半径↓（一般） |
| | 同主族：上→下，金属性↑、非金属性↓、原子半径↑ |

#### 4.2 原子结构

| 知识点 | 要点 |
|--------|------|
| 原子构成 | 质子、中子、电子；质量数 A = 质子数 + 中子数 |
| 同位素 | 质子数相同、中子数不同；如 ¹H、²H、³H |
| 相对原子质量 | 按同位素丰度加权平均 |

#### 4.3 核外电子排布

| 知识点 | 要点 |
|--------|------|
| 电子层 | K、L、M、N…；每层最多 2n² 个电子 |
| 排布规律 | 2、8、8…（前 20 号元素）；最外层不超过 8（H、He 为 2） |
| 化学性质 | 最外层电子数决定得失电子能力 |

#### 4.4 化学键

| 类型 | 要点 |
|------|------|
| 离子键 | 阴阳离子静电作用；活泼金属 + 活泼非金属 |
| 共价键 | 共用电子对；非金属单质、共价化合物 |
| 分子间作用力 | 比化学键弱；影响熔沸点 |

**易错点：** 含共价键的化合物不一定是共价化合物（如 NH₄Cl 含离子键）

## 四、核心反应类型

```
化学反应
├── 氧化还原反应（第 2 章系统学习）
│   ├── 置换反应
│   ├── 化合/分解中的升降价
│   └── 歧化反应（Cl₂、NO₂ 等）
├── 离子反应（第 2 章系统学习）
│   ├── 复分解型（沉淀/气体/弱电解质）
│   └── 氧化还原型（Cl₂ + 2I⁻ → I₂ + 2Cl⁻）
└── 非氧化还原
    ├── 化合、分解（无升降价）
    └── 中和反应
```

## 五、化学计算

| 类型 | 方法 |
|------|------|
| 物质的量换算 | n = m/M = N/N_A = V/22.4（标况气体） |
| 溶液浓度 | c = n/V；稀释 c₁V₁ = c₂V₂ |
| 化学方程式计算 | 找关系式 → 列比例 → 注意过量判断 |
| 气体计算 | 同温同压 V₁/V₂ = n₁/n₂ |
| 纯度 / 产率 | 实际量 / 理论量 × 100% |
