## 二、焰色反应

**定义：** 某些金属或它们的化合物在灼烧时，火焰呈现特殊颜色。

**操作要点：** 用铂丝（或铁丝）蘸取样品，在酒精灯火焰上灼烧观察。

| 金属元素 | 焰色 | 记忆提示 |
|----------|------|----------|
| 钠（Na） | **黄色** | 钠黄 |
| 钾（K） | **紫色** | 钾紫（须透过蓝色钴玻璃观察，滤去钠的黄色干扰） |
| 钙（Ca） | **砖红色** | 钙砖红 |
| 铜（Cu） | **绿色** | 铜绿 |
| 钡（Ba） | **黄绿色** | 钡黄绿 |
| 锶（Sr） | **洋红色** | 锶洋红 |
| 锂（Li） | **紫红色** | 锂紫红 |

**初中必记四种：** Na 黄、K 紫（钴玻璃）、Ca 砖红、Cu 绿

**注意区分：**
- **焰色反应** ≠ 燃烧火焰颜色（如 H₂ 淡蓝火焰）
- 焰色反应是**元素性质**，用于检验金属离子
- 铁丝灼烧本身无焰色，可代替铂丝使用

## 三、燃烧时的火焰颜色

| 物质 | 火焰颜色 | 备注 |
|------|----------|------|
| 氢气 H₂ | 淡蓝色 | 点燃前须验纯 |
| 一氧化碳 CO | 蓝色 | 燃烧产物 CO₂ |
| 甲烷 CH₄ | 淡蓝色 | 天然气、沼气 |
| 乙醇 C₂H₅OH | 淡蓝色 | 酒精灯火焰 |
| 硫 S（在 O₂ 中） | 蓝紫色 | 生成 SO₂ |
| 磷 P（在 O₂ 中） | — | 产生大量白烟（P₂O₅） |
| 镁 Mg | 耀眼白光 | 生成白色 MgO |
| 铁 Fe | 火星四射 | 生成黑色 Fe₃O₄ |
| 铜 Cu | — | 表面变黑（生成 CuO） |
| 木炭 C | 红光/白光 | 充分燃烧 CO₂ |
| 石蜡 / 蜡烛 | 黄色 | 外焰温度最高 |
| 酒精灯外焰 | 淡蓝色 | 外焰温度最高 |

## 四、常见酸的颜色

### 4.1 酸溶液（通常状态）

| 酸 | 颜色 / 状态 | 其他特征 |
|----|-------------|----------|
| 盐酸 HCl | **无色**液体 | 刺激性气味，浓盐酸有白雾（挥发性） |
| 硫酸 H₂SO₄ | **无色**油状液体 | 浓硫酸吸水、脱水 |
| 硝酸 HNO₃ | 浓硝酸呈**黄色** | 久置变黄（溶解 NO₂）；稀硝酸无色 |
| 碳酸 H₂CO₃ | 无色 | 不稳定，常温分解 |
| 醋酸 CH₃COOH | 无色 | 食醋有酸味 |
| 磷酸 H₃PO₄ | 无色 | — |

### 4.2 酸相关记忆

```
无色酸：HCl、H₂SO₄、稀 HNO₃、H₂CO₃、CH₃COOH
黄色酸：浓 HNO₃（久置或见光分解产生 NO₂）
```

## 五、常见碱的颜色

### 5.1 碱溶液

| 碱 | 颜色 | 状态 / 备注 |
|----|------|-------------|
| 氢氧化钠 NaOH | **无色** | 固体白色，易潮解 |
| 氢氧化钙 Ca(OH)₂ | **无色** | 石灰水、石灰浆 |
| 氢氧化钾 KOH | **无色** | 与 NaOH 相似 |
| 氨水 NH₃·H₂O | **无色** | 刺激性气味 |

### 5.2 难溶碱（沉淀）

| 碱 | 颜色 | 备注 |
|----|------|------|
| 氢氧化铜 Cu(OH)₂ | **蓝色** | 不溶于水 |
| 氢氧化铁 Fe(OH)₃ | **红褐色** | 不溶于水 |
| 氢氧化亚铁 Fe(OH)₂ | **白色** | 迅速变灰绿，最终变红褐（被氧化） |
| 氢氧化镁 Mg(OH)₂ | **白色** | 不溶于水 |
| 氢氧化铝 Al(OH)₃ | **白色** | 两性，溶于酸和强碱 |

### 5.3 碱相关记忆

```
可溶性碱溶液：NaOH、KOH、Ca(OH)₂、NH₃·H₂O → 均无色
蓝色碱：Cu(OH)₂（唯一蓝色沉淀碱）
红褐色碱：Fe(OH)₃
白色碱沉淀：Mg(OH)₂、Al(OH)₃、Fe(OH)₂
```

## 六、常见盐的颜色

### 6.1 盐溶液颜色

| 盐 | 溶液颜色 | 备注 |
|----|----------|------|
| NaCl、KCl、Na₂SO₄ 等 | **无色** | 大多数盐溶液无色 |
| 硫酸铜 CuSO₄ | **蓝色** | 特征明显 |
| 氯化铜 CuCl₂ | **蓝色** | 浓溶液蓝绿 |
| 硫酸亚铁 FeSO₄ | **浅绿色** | Fe²⁺ |
| 氯化亚铁 FeCl₂ | **浅绿色** | Fe²⁺ |
| 氯化铁 FeCl₃ | **黄色** | Fe³⁺，浓溶液棕黄色 |
| 硫酸铁 Fe₂(SO₄)₃ | **黄色** | Fe³⁺ |
| 高锰酸钾 KMnO₄ | **紫红色** | 强氧化性 |
| 碘化钾与淀粉 | **蓝色** | 淀粉遇碘变蓝（非盐本身） |

### 6.2 盐固体颜色

| 盐 | 固体颜色 | 俗称 / 备注 |
|----|----------|-------------|
| NaCl | 白色 | 食盐 |
| CaCO₃ | 白色 | 石灰石、大理石 |
| Na₂CO₃ | 白色 | 纯碱 |
| NaHCO₃ | 白色 | 小苏打 |
| CuSO₄（无水） | **白色** | 吸水变蓝 |
| CuSO₄·5H₂O | **蓝色** | 胆矾、蓝矾 |
| FeSO₄·7H₂O | **浅绿色** | 绿矾 |
| KMnO₄ | **暗紫色** | 灰锰氧 |
| K₂MnO₄ | **墨绿色** | 制 O₂ 时生成 |
| AgCl | **白色** | 不溶于酸 |
| BaSO₄ | **白色** | 不溶于酸 |
| Cu₂(OH)₂CO₃ | **绿色** | 铜绿 |

### 6.3 酸碱盐颜色口诀

```
酸液多数无色状，浓硝久置呈黄色。
碱液可溶皆无色，Cu(OH)₂蓝、Fe(OH)₃红褐记心上。
盐液蓝色铜离子，浅绿亚铁黄铁离子；
紫红高锰酸钾液，其余多数无色液。
```

## 八、沉淀颜色速记

| 颜色 | 常见沉淀 | 是否溶于酸 |
|------|----------|------------|
| **白色** | AgCl、BaSO₄、CaCO₃、Mg(OH)₂、Al(OH)₃、BaCO₃ | AgCl、BaSO₄ 不溶；其余多数溶 |
| **蓝色** | Cu(OH)₂ | 溶于酸 |
| **红褐色** | Fe(OH)₃ | 溶于酸 |
| **灰绿色** | Fe(OH)₂ 中间产物 | — |
| **黑色** | CuS、Ag₂S | — |

### 检验相关沉淀

| 检验对象 | 所用试剂 | 现象 |
|----------|----------|------|
| Cl⁻ | AgNO₃ + HNO₃ | 白色沉淀 AgCl |
| SO₄²⁻ | BaCl₂ + HCl | 白色沉淀 BaSO₄ |
| CO₃²⁻ | 稀盐酸 | 气泡（CO₂） |
| CO₂ | 澄清石灰水 | 白色浑浊 |
| NH₄⁺ | NaOH 加热 | 刺激性气体，湿润石蕊试纸变蓝 |
| Fe³⁺ | NaOH | 红褐色沉淀 |
| Cu²⁺ | NaOH | 蓝色沉淀 |
