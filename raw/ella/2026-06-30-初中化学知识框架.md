## 二、化学基本概念

### 2.1 走进化学世界

| 知识点 | 要点 |
|--------|------|
| 化学研究对象 | 物质及其变化 |
| 物理变化 vs 化学变化 | 是否有**新物质**生成 |
| 物理性质 vs 化学性质 | 是否需发生化学变化才能体现 |
| 实验基本操作 | 取用、加热、过滤、蒸发、气密性检查 |
| 常用仪器 | 试管、烧杯、集气瓶、酒精灯、量筒、天平 |

**探究性实验步骤：** 提出问题 → 猜想假设 → 设计实验 → 收集证据 → 得出结论 → 反思评价

### 2.2 物质分类

```
物质
├── 纯净物
│   ├── 单质
│   │   ├── 金属单质（Fe、Cu、Al…）
│   │   └── 非金属单质（O₂、C、S…）
│   └── 化合物
│       ├── 氧化物（CO₂、Fe₂O₃…）
│       ├── 酸（HCl、H₂SO₄…）
│       ├── 碱（NaOH、Ca(OH)₂…）
│       └── 盐（NaCl、Na₂CO₃…）
└── 混合物（空气、溶液、合金、石油…）
```

### 2.3 化学用语

| 内容 | 掌握要求 |
|------|----------|
| 元素符号 | H、O、C、N、S、P、Fe、Cu、Ag、Al、Mg、Zn、Cl 等 |
| 化学式 | 读写、含义（元素种类、原子个数比）、化合价 |
| 化学方程式 | 书写原则、配平、化学意义、计算应用 |
| 离子符号 | H⁺、OH⁻、Na⁺、Cl⁻、CO₃²⁻、SO₄²⁻、NH₄⁺ 等 |

**化合价口诀（部分）：** 一价氢氯钾钠银，二价氧钙钡镁锌，三铝四硅五价磷，二三铁、二四碳，二四六硫都齐全，铜汞二价最常见。

### 2.4 微观粒子

| 粒子 | 定义与特点 |
|------|------------|
| 分子 | 保持物质化学性质的最小粒子；不断运动、有间隔 |
| 原子 | 化学变化中最小粒子；质子数 = 核外电子数 = 原子序数 |
| 离子 | 带电的原子或原子团（阳离子、阴离子） |

**关系：** 分子由原子构成；化学变化中分子可分、原子不可分。

## 三、分单元知识框架

### 第一单元：走进化学世界

- 化学变化与物理变化的判断
- 实验安全与基本操作规范
- 探究性实验的一般思路

### 第二单元：空气和氧气

#### 空气

| 内容 | 要点 |
|------|------|
| 成分 | N₂（约 78%）、O₂（约 21%）、稀有气体、CO₂ 等 |
| 测定氧气含量 | 红磷燃烧法：进入水的体积 ≈ O₂ 体积 |
| 空气污染 | SO₂、NO₂、可吸入颗粒物；酸雨、臭氧层破坏 |
| 防治措施 | 清洁能源、尾气处理、植树造林 |

#### 氧气

| 类别 | 内容 |
|------|------|
| 物理性质 | 无色无味气体，密度略大于空气 |
| 化学性质 | 助燃性（氧化性）；与 C、S、P、Fe 等反应 |
| 实验室制取 | 加热 KMnO₄ / 分解 H₂O₂ / 加热 KClO₃ |
| 工业制取 | 分离液态空气（物理变化） |
| 用途 | 医疗急救、炼钢、气焊、航天 |

**制取方程式：**

```
2KMnO₄  →  K₂MnO₄ + MnO₂ + O₂↑        （加热）
2H₂O₂   →  2H₂O + O₂↑                 （MnO₂ 催化）
2KClO₃  →  2KCl + 3O₂↑                （MnO₂ 催化，加热）
```

**验满：** 带火星木条放在集气瓶**口**，复燃则满。

### 第三单元：水和氢

| 主题 | 核心内容 |
|------|----------|
| 水的组成 | 电解水：H₂ 与 O₂ 体积比 2:1 → 水由 H、O 元素组成 |
| 水的净化 | 沉淀 → 过滤 → 吸附（活性炭）→ 蒸馏（净化程度最高） |
| 硬水与软水 | 肥皂水区分；煮沸、蒸馏可软化硬水 |
| 氢气 | 最轻气体；可燃性（纯净氢气点燃前须验纯）；还原性 |
| 制取氢气 | Zn + H₂SO₄ → ZnSO₄ + H₂↑（排水法或向下排空气法收集） |

### 第四单元：碳和碳的氧化物

#### 碳单质

| 物质 | 结构 | 性质与用途 |
|------|------|------------|
| 金刚石 | 正四面体网状 | 硬度大、折光性好 → 钻头、饰品 |
| 石墨 | 层状结构 | 导电、润滑 → 电极、铅笔芯 |
| C₆₀ | 分子结构 | 新材料研究 |

**化学性质：** 可燃性（充分燃烧 CO₂，不充分 CO）；还原性（与 CuO、Fe₂O₃ 反应）

#### 二氧化碳 CO₂

| 类别 | 内容 |
|------|------|
| 物理性质 | 无色无味，密度大于空气，能溶于水 |
| 化学性质 | 不燃烧不支持燃烧；与水反应；与碱反应 |
| 实验室制取 | CaCO₃ + 2HCl → CaCl₂ + H₂O + CO₂↑ |
| 验满 | 燃着的木条放在瓶口，熄灭则满 |
| 检验 | 通入澄清石灰水，变浑浊 |
| 用途 | 灭火、干冰、光合作用原料、化工原料 |
| 环境影响 | 温室气体，导致全球变暖 |

#### 一氧化碳 CO

- 无色无味有毒气体，与血红蛋白结合
- 可燃性、还原性（炼铁：CO + Fe₂O₃ → Fe + CO₂）
- 注意：CO 燃烧火焰呈蓝色

### 第五单元：金属

#### 金属的物理性质

导电性、导热性、有金属光泽、延展性、密度硬度各异

#### 金属的化学性质

1. 与氧气反应（活泼性不同，现象不同）
2. 与稀酸反应（置换反应，排在 H 前面的金属）
3. 与盐溶液反应（活泼金属置换不活泼金属）

#### 金属活动性顺序

```
K  Ca  Na  Mg  Al  Zn  Fe  Sn  Pb  (H)  Cu  Hg  Ag  Pt  Au
←—————————— 越靠前越活泼 ——————————→
```

**应用：**
- 排在 H 前可与稀酸反应产生 H₂
- 前置换后：Fe + CuSO₄ → FeSO₄ + Cu
- K、Ca、Na 太活泼，先与水反应

#### 铁的冶炼

```
3CO + Fe₂O₃  →  2Fe + 3CO₂    （高温，还原反应）
```

#### 金属资源保护

- 防锈：刷漆、涂油、电镀、制成不锈钢
- 回收废旧金属
- 合理开采、寻找替代材料

### 第六单元：溶液

| 概念 | 要点 |
|------|------|
| 溶液组成 | 溶质 + 溶剂（水是最常用溶剂） |
| 溶解 | 物理过程；分子（或离子）扩散 |
| 乳化 | 乳化剂使油分散成细小液滴 |
| 饱和溶液 | 一定温度、一定量溶剂中不能再溶解该溶质的溶液 |
| 溶解度 | 100 g 水中最多溶解的质量（g）；受温度影响 |
| 溶质质量分数 | ω = 溶质质量 / 溶液质量 × 100% |

**配制一定溶质质量分数溶液：** 计算 → 称量（量取）→ 溶解 → 装瓶贴标签

**溶解度曲线应用：**
- 查溶解度数值
- 判断饱和状态
- 选择结晶方法（蒸发结晶 / 降温结晶）

### 第七单元：酸和碱

#### 酸（含 H⁺）

| 常见酸 | HCl、H₂SO₄、HNO₃、H₂CO₃ |
|--------|-------------------------|
| 通性 | ① 使紫色石蕊变红 ② 与活泼金属 ③ 与碱 ④ 与某些盐 |
| 浓 H₂SO₄ | 吸水性（干燥剂）、脱水性；稀释：**酸入水**，沿器壁缓慢倒入 |

#### 碱（含 OH⁻）

| 常见碱 | NaOH、Ca(OH)₂、KOH |
|--------|---------------------|
| 通性 | ① 使紫色石蕊变蓝、酚酞变红 ② 与酸 ③ 与某些盐 |
| 特性 | NaOH 易潮解、有腐蚀性；Ca(OH)₂ 俗称石灰水，微溶 |

#### 中和反应

```
酸 + 碱 → 盐 + 水    （放热反应）
例：HCl + NaOH → NaCl + H₂O
```

**应用：** 改良酸性土壤、处理工厂废水、医药（胃药）

### 第八单元：盐

| 物质 | 俗称 | 性质与应用 |
|------|------|------------|
| NaCl | 食盐 | 调味品、化工原料、腌制 |
| Na₂CO₃ | 纯碱、苏打 | 与酸反应、玻璃工业 |
| NaHCO₃ | 小苏打 | 受热分解、焙烤食品 |
| CaCO₃ | 石灰石、大理石 | 建筑材料、制 CO₂ |
| CuSO₄ | — | 波尔多液、湿法炼铜 |
| Ca(OH)₂ | 熟石灰 | 改良酸性土壤、建筑材料 |

#### 复分解反应

```
AB + CD → AD + CB
```

**发生条件（满足其一）：** 生成水、气体或沉淀

### 第九单元：化学与生活

| 主题 | 要点 |
|------|------|
| 有机物 | 含碳化合物（除 CO、CO₂、碳酸盐、碳酸等） |
| 三大营养素 | 糖类（能量）、油脂（能量）、蛋白质（构成细胞） |
| 维生素与微量元素 | 调节代谢；缺乏症 |
| 材料 | 塑料（热塑/热固）、合成纤维、合成橡胶 |
| 能源 | 化石燃料（煤、石油、天然气）；新能源（氢能、太阳能等） |
| 环境保护 | 污水、废气、固体废弃物处理；绿色化学 |

## 四、化学反应类型

| 类型 | 定义 | 通式 | 典型例子 |
|------|------|------|----------|
| 化合反应 | 多变一 | A + B → C | C + O₂ → CO₂ |
| 分解反应 | 一变多 | A → B + C | 电解水、制 O₂ |
| 置换反应 | 单质 + 化合物 | A + BC → AC + B | Zn + H₂SO₄ |
| 复分解反应 | 化合物互换 | AB + CD → AD + CB | HCl + NaOH |
| 氧化反应 | 得氧 / 失电子 | — | 燃烧、缓慢氧化 |
| 还原反应 | 失氧 / 得电子 | — | CO 还原 Fe₂O₃ |

**燃烧条件：** 可燃物 + 氧气 + 温度达到着火点（灭火的原理：破坏任一条件）

## 五、化学计算

### 5.1 基本计算

- 相对分子质量 = 各原子相对原子质量之和
- 元素质量比 = 各元素相对原子质量 × 原子个数之比
- 元素质量分数 = 该元素相对原子质量 × 原子个数 / 相对分子质量 × 100%

### 5.2 化学方程式计算

**步骤：** 设未知数 → 写化学方程式 → 标相对质量 → 列比例式 → 求解 → 答

**依据：** 质量守恒定律（反应前后各原子种类、数目、质量不变）

### 5.3 溶液计算

- 溶质质量 = 溶液质量 × 溶质质量分数
- 稀释前后溶质质量不变：m₁ω₁ = m₂ω₂
- 与化学方程式结合：先求溶质质量，再换算

### 5.4 图表题

- **溶解度曲线：** 查溶解度、判断饱和、结晶方法
- **反应曲线：** 质量/体积随时间变化（如制 O₂、中和反应 pH 变化）

| 操作 | 对溶质质量分数的影响 |

## 七、物质鉴别与颜色

### 7.1 常见颜色

| 颜色/现象 | 常见物质 |
|-----------|----------|
| 黑色固体 | CuO、Fe₃O₄、C、MnO₂ |
| 红棕色固体 | Fe₂O₃ |
| 红色固体 | Cu |
| 蓝色溶液 | CuSO₄、CuCl₂ |
| 绿色溶液 | FeSO₄、FeCl₂ |
| 黄色溶液 | FeCl₃ |
| 紫色溶液 | KMnO₄ |
| 白色沉淀 | BaSO₄、AgCl、CaCO₃ |
| 红褐色沉淀 | Fe(OH)₃ |
| 蓝色沉淀 | Cu(OH)₂ |

### 7.2 常见气体鉴别

| 气体 | 鉴别方法 |
|------|----------|
| O₂ | 带火星木条复燃 |
| CO₂ | 澄清石灰水变浑浊 |
| H₂ | 点燃，发出尖锐爆鸣声（验纯） |
| CO | 点燃后通入澄清石灰水变浑浊 |
