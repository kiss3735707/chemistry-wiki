## 一、为什么学习物质的量

化学研究既涉及**宏观**可称量的物质（质量 g），也涉及**微观**粒子（原子、分子、离子）。1 滴水约含 10²¹ 个水分子，用"个"计数不现实。

**物质的量**是国际单位制中的基本物理量之一，单位 **mol（摩尔）**，是连接宏观与微观的桥梁。

## 二、核心概念（沪科版 1.2）

### 2.1 物质的量 n 与阿伏加德罗常数 N_A

| 项目 | 内容 |
|------|------|
| 定义 | 表示含有一定数目粒子的集合体 |
| 符号 | n |
| 单位 | mol（摩尔） |
| N_A | 阿伏加德罗常数，N_A ≈ 6.02×10²³ mol⁻¹ |
| 含义 | 1 mol 任何粒子（原子、分子、离子、电子等）所含粒子数目**相同** |

**公式：** `n = N / N_A`

**规范表述：**
- "1 mol H₂O" 指 6.02×10²³ 个水分子
- 0.2 mol H₂ 含 **0.4 mol** H 原子（不是 0.2 mol H 原子）

### 2.2 摩尔质量 M

| 项目 | 内容 |
|------|------|
| 定义 | 单位物质的量的物质所具有的质量 |
| 公式 | M = m / n |
| 单位 | g/mol |
| **重要规律** | M 在**数值上**等于相对原子质量或相对分子质量 |

**公式：** `n = m / M`

| 物质 | 相对分子/原子质量 | M |
|------|-------------------|---|
| Fe | 56 | 56 g/mol |
| H₂O | 18 | 18 g/mol |
| CO₂ | 44 | 44 g/mol |
| NaOH | 40 | 40 g/mol |
| H₂SO₄ | 98 | 98 g/mol |

### 2.3 气体摩尔体积 V_m

| 项目 | 内容 |
|------|------|
| 定义 | 单位物质的量的气体在**一定温度和压强下**所占的体积 |
| 标准状况（标况） | 0 ℃、101 kPa |
| 标况下 | V_m ≈ **22.4 L/mol** |
| **适用范围** | **仅气体**；液体、固体不用 22.4 |

**公式：** `n = V / V_m`（标况，V 单位 L）

**规律：** 同温同压下，相同物质的量的不同气体，体积**相同**（阿伏加德罗定律）。

### 2.4 物质的量浓度 c

| 项目 | 内容 |
|------|------|
| 定义 | 单位体积溶液中所含溶质的物质的量 |
| 公式 | c = n / V |
| 单位 | mol/L |
| **注意** | V 是**溶液体积**（L），不是溶剂体积 |

**稀释定律：** 稀释前后溶质物质的量不变 → **c₁V₁ = c₂V₂**

### 2.5 换算关系总图（沪科版核心）

```
                    ┌──×M──→ 质量 m（g）
                    │         │
                    │         └──÷M──┐
                    │                 ↓
粒子数 N ──÷N_A──→ 物质的量 n ──×N_A──→ 粒子数 N
                    │                 ↑
                    │         ┌──×22.4──┘
                    └──÷22.4──→ 气体体积 V（L，标况）
                    │
                    └──÷V──→ 物质的量浓度 c（mol/L）
```

**核心公式链：**

```
n = m/M = N/N_A = V/22.4（标况气体）
c = n/V；稀释 c₁V₁ = c₂V₂
```

## 三、计算题型与例题（沪科版常考）

### 类型 1：粒子数 ↔ 物质的量

**例 1** 0.5 mol H₂SO₄ 中含 H、S、O 原子的物质的量各为多少？

| 粒子 | 计算 | 结果 |
|------|------|------|
| H | 0.5 × 2 | 1 mol |
| S | 0.5 × 1 | 0.5 mol |
| O | 0.5 × 4 | 2 mol |

**例 2** 6.02×10²² 个 Na⁺ 的物质的量？  
n = 6.02×10²² / 6.02×10²³ = **0.1 mol**

### 类型 2：质量 ↔ 物质的量

**例 3** 88 g CO₂ 的物质的量？  
n = 88/44 = **2 mol**

**例 4** 配制 0.1 mol/L NaOH 250 mL，需 NaOH 固体多少克？  
n = 0.1 × 0.25 = 0.025 mol → m = 0.025 × 40 = **1.0 g**

### 类型 3：标况气体体积

**例 5** 标况下 33.6 L CO₂ 的物质的量？  
n = 33.6/22.4 = **1.5 mol**

**例 6** 标况下 5.6 g 铁与足量稀 H₂SO₄ 反应，生成 H₂ 体积？  
Fe + H₂SO₄ → FeSO₄ + H₂↑  
n(Fe) = 5.6/56 = 0.1 mol → n(H₂) = 0.1 mol → V = **2.24 L**

### 类型 4：溶液浓度与稀释

**例 7** 将 20 mL 5 mol/L HCl 稀释至 500 mL，求 c₂。  
c₁V₁ = c₂V₂ → 5 × 0.02 = c₂ × 0.5 → c₂ = **0.2 mol/L**

### 类型 5：混合气体（同温同压）

**例 8** 标况下 11.2 L H₂ 与 22.4 L O₂ 混合，总体积？  
n(H₂)=0.5 mol，n(O₂)=1 mol → n总=1.5 mol → V总 = **33.6 L**

**规律：** V₁/V₂ = n₁/n₂（同温同压）

### 类型 6：综合大题（方程式 + 多步换算）

**例 9** 某气体在标况下密度 1.96 g/L，求其相对分子质量。  
M = 22.4 × ρ = 22.4 × 1.96 ≈ **44**（CO₂）

## 四、沪科版实验衔接（1.2 活动 + 1.3 配制）

### 4.1 活动：测定气体摩尔体积

**原理：** Mg + 2HCl → MgCl₂ + H₂↑，测 H₂ 体积，求 V_m = V/n

**要点：**
- 检查装置气密性
- 冷却至室温再读气体体积
- 注意水蒸气对体积的影响

### 4.2 配制一定物质的量浓度的溶液（1.3）

**步骤：** 计算 → 称量（或量取）→ 溶解 → 转移 → 洗涤 → **定容** → 摇匀 → 贴标签

**仪器：** 天平、烧杯、玻璃棒、容量瓶、胶头滴管、量筒

**误差分析（对 c 的影响）：**

| 操作 | 结果 |
|------|------|
| 定容时**仰视**凹液面 | c **偏小** |
| 定容时**俯视** | c **偏大** |
| 未洗涤烧杯和玻璃棒 | c **偏小** |
| 容量瓶内壁附有水珠（未干燥） | **无影响** |

## 五、易错点汇总（沪科版期末高频）

### 概念辨析

| 说法 | 正误 |
|------|------|
| 摩尔是物质的质量单位 | ❌ 是物质的量的单位 |
| 1 mol 任何物质质量相同 | ❌ 质量不同（M 不同） |
| 1 mol 任何物质粒子数相同 | ✅ 均为 N_A 个 |
| 标况 22.4 L H₂O = 1 mol | ❌ 水是液体 |
| 64 g O₂ 含 2 mol 氧原子 | ❌ 是 2 mol O₂ 分子 = 4 mol O 原子 |

### 化学式与粒子关系

| 1 mol … | 含 … |
|---------|------|
| H₂ | 1 mol 分子，**2 mol** H 原子 |
| H₂SO₄ | 2 mol H、1 mol S、4 mol O 原子 |
| NaCl（晶体） | 1 mol Na⁺ + 1 mol Cl⁻ |

### 浓度陷阱

- c = n/V 中 V 必须是**溶液体积**
- 1 mol NaCl 溶于 1 L 水 → c **≠** 1 mol/L
